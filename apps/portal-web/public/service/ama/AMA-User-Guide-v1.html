Organization Structure / 조직 구조

AMA supports a matrix organization structure with the following hierarchy:

AMA는 아래와 같은 계층의 매트릭스 조직 구성을 지원합니다.

| Level / 계층 | English | Korean / 한국어 | Description / 설명 |

|------|---------|---------|------|

| Entity | Entity (Company) | 법인 (회사) | Top-level legal entity. All data is isolated per entity. / 최상위 법인 단위. 모든 데이터가 법인별로 격리됩니다. |

| Unit | Unit (Department) | 유닛 (부서) | A person belongs to one unit. Units can have sub-units (Section). / 1명의 개인은 하나의 유닛(부서)에 소속됩니다. 유닛은 하위 유닛(섹션)을 만들 수 있습니다. |

| Cell | Cell (Project Team) | 셀 (프로젝트팀) | A person can belong to multiple cells. Similar to Task Force Teams in typical companies. / 1명의 개인은 상황에 따라 여러 셀에 소속될 수 있습니다. 일반 회사의 TF(Task Force Team) 형식입니다. |

> Matrix Organization / 매트릭스 조직: Unit (vertical, permanent structure) and Cell (horizontal, project-based) work together to form a flexible organization.

> Unit(수직적·상설 조직)과 Cell(수평적·프로젝트 기반)이 함께 작동하여 유연한 조직 구성을 지원합니다.

로그인 직후 표시되는 통합 대시보드 페이지입니다. 오늘 처리해야 할 미션, 작업, 이슈, 일정, 회의록을 한 화면에서 확인할 수 있습니다.

1. 할일 TODO

> Path / 경로: Sidebar > TODO / 사이드바 > 할일 (/todos)

1.1 Overview / 개요

Kanban-based tool for managing personal and unit tasks. Auto-classified into 3 columns (Done / Today / Upcoming) based on due dates. View scope: personal, group, or company-wide.

개인 및 유닛(부서) 단위의 할일을 관리하는 칸반 기반 도구입니다. 마감일 기준으로 완료 / 오늘 / 예정의 3개 컬럼으로 자동 분류되며, 개인 · 그룹 · 전사 범위로 할일을 조회할 수 있습니다.

1.2 Screen Layout / 화면 구성

| Area / 영역 | Description / 설명 |

|------|------|

| Tab Bar / 탭 바 | 3 tabs for scope: My Tasks · Group · All / 내 할일 · 그룹 · 전사 3개 탭으로 조회 범위 전환 |

| Kanban Board / 칸반 보드 | 3 columns — Done(green), Today(blue), Upcoming(orange) / 3개 컬럼 — 완료(초록), 오늘(파랑), 예정(주황) |

| Task Card / 할일 카드 | Title, due date, status badge, tags, participant avatars / 제목, 마감일, 상태 배지, 태그, 참여자 아바타 표시 |

| + Add Task / + 할일 추가 | Top-right button (My Tasks tab only) / 우상단 버튼 (내 할일 탭에서만 표시) |

1.3 Key Features / 주요 기능

Create Task / 할일 생성

  1. Click [+ Add Task / + 할일 추가] button at top-right / 우상단 버튼 클릭
  2. Input fields / 입력 항목:
  • Title / 제목 (required / 필수)
  • Description / 설명 (optional / 선택, 텍스트)
  • Due Date / 마감일 (required / 필수, 날짜)
  • Tags / 태그 (optional, comma-separated / 선택, 쉼표로 구분)
  • Linked Issue / 연결 이슈 (optional / 선택, 기존 이슈 연결)
  • Participants / 참여자 (optional, multi-select / 선택, 멤버 다중 선택)
  1. Auto-translation dialog appears on save (KO→EN/VN) / 저장 시 자동 번역 대화상자가 표시됩니다

Status Management / 상태 관리

Click the check icon on a task card to cycle status. / 할일 카드의 체크 아이콘을 클릭하면 상태가 순환됩니다.

| Status / 상태 | Meaning / 의미 | Badge Color / 배지 색상 |

|------|------|-----------|

| SCHEDULED | Scheduled / 예정 | Orange / 주황 |

| IN_PROGRESS | In Progress / 진행중 | Blue / 파랑 |

| COMPLETED | Completed / 완료 | Green / 초록 |

Auto-Classification Rules / 자동 분류 규칙

  • Done column / 완료 컬럼: Tasks with COMPLETED status / 상태가 COMPLETED인 할일
  • Today column / 오늘 컬럼: Incomplete tasks due today or overdue / 마감일이 오늘 이하인 진행중/예정 할일
  • Upcoming column / 예정 컬럼: Incomplete tasks due tomorrow or later / 마감일이 내일 이후인 미완료 할일

Detail / Edit / Delete / 상세보기 · 수정 · 삭제

  • Card click → Detail Modal (title, description, status, participants, linked issues) / 카드 클릭 → 상세 모달 표시
  • Card menu → Edit / Delete / 카드 메뉴 → 수정 / 삭제 선택
  • Status change history available (history icon click) / 상태 변경 이력 조회 가능

Group / All Tabs / 그룹 · 전사 탭

  • Group tab: View tasks of same unit members / 같은 유닛(부서) 소속 멤버들의 할일 조회
  • All tab: View all company tasks / 전체 회사의 할일 조회
  • Read-only; cannot edit others' tasks / 조회 전용이며, 다른 사람의 할일을 수정할 수 없습니다

2. 노트 Meeting Notes

> Path / 경로: Sidebar > Meeting Notes / 사이드바 > 노트 (/meeting-notes)

2.1 Overview / 개요

Tool for creating and managing two types of notes: Meeting Notes and Memos. Set visibility scope from private memos to company-wide meeting minutes.

회의록(Meeting Note)과 메모(Memo) 두 가지 유형의 노트를 작성·관리하는 도구입니다. 공개 범위를 설정하여 개인 메모부터 전사 공유 회의록까지 다양한 용도로 활용할 수 있습니다.

2.2 Screen Layout / 화면 구성

| Area / 영역 | Description / 설명 |

|------|------|

| Type Filter / 유형 필터 | All · Meeting Note · Memo buttons / 전체 · 회의록 · 메모 버튼 |

| Scope Filter / 공개범위 필터 | All · Private · Group · Entity · Public / 전체 · 비공개 · 그룹 · 법인 · 공개 |

| Card Grid / 카드 그리드 | 2-column card layout (mobile: 1 column) / 2열 카드 레이아웃 (모바일 1열) |

| + New Note / + 새 노트 | Create button at top-right / 우상단 생성 버튼 |

2.3 Key Features / 주요 기능

Create Note / 노트 생성

  1. Click [+ New Note / + 새 노트] button / 버튼 클릭
  2. Input fields / 입력 항목:
  • Type / 유형: Meeting Note / Memo / 회의록 / 메모
  • Title / 제목 (required / 필수)
  • Content / 내용 (rich text editor, HTML formatting / 리치 텍스트 에디터, HTML 서식 지원)
  • Meeting Date / 회의 일자 (shown for Meeting Note type only / 회의록 유형 시 표시)
  • Visibility / 공개 범위: PRIVATE / GROUP / ENTITY / PUBLIC
  • Unit / 유닛(부서) (when GROUP scope selected / GROUP 범위 선택 시)
  1. Displayed as card in the list after saving / 저장 후 목록에 카드로 표시됩니다

Visibility Scope / 공개 범위

| Scope / 범위 | Audience / 조회 대상 |

|------|-----------|

| PRIVATE | Author only / 작성자 본인만 |

| GROUP | Designated unit/group members / 지정된 유닛(부서)/그룹 멤버 |

| ENTITY | All members in same entity / 같은 법인 소속 전체 |

| PUBLIC | Company-wide / 전사 공개 |

Detail Page / 상세 페이지

  • Card click → Detail page (/meeting-notes/:id) / 카드 클릭 → 상세 페이지로 이동
  • HTML formatted note content rendering / HTML 형식 노트 내용 렌더링
  • Translation Panel / 번역 패널: Auto-translation support (KO/EN/VN) / 다국어 자동 번역 지원
  • Edit / Delete buttons (author only) / 수정 / 삭제 버튼 (작성자만 표시)

3. 출근예정표 Attendance

> Path / 경로: Sidebar > Attendance / 사이드바 > 출근예정표 (/attendance)

3.1 Overview / 개요

Tool to view and register unit members' weekly work schedules at a glance. Displayed in a weekly table format showing each member's Mon–Sun work types and hours.

유닛(부서) 멤버들의 주간 근무 일정을 한 눈에 확인하고 등록하는 도구입니다. 주간 단위 표 형식으로 각 멤버의 월~일(7일) 근무 유형과 시간이 표시됩니다.

3.2 Screen Layout / 화면 구성

| Area / 영역 | Description / 설명 |

|------|------|

| Week Navigation / 주간 네비게이션 | < > buttons for prev/next week. Default: next Monday / 이전/다음 주 이동. 기본은 다음 주 월요일 |

| Schedule Table / 스케줄 테이블 | Rows: members, Cols: Mon–Sun. Own row always on top / 행: 멤버, 열: 월~일. 본인이 항상 최상단 |

| Cell Content / 셀 내용 | Work type badge + start–end time / 근무유형 배지 + 시작~종료 시간 |

| Edit Form / 편집 폼 | Shown at bottom on cell click / 셀 클릭 시 하단에 근무유형·시작시간 선택 폼 |

| NOTE Column / 비고 컬럼 | Remarks column (e.g., remote work days) / 비고란 (재택 근무일 등 표시) |

3.3 Work Types / 근무 유형

| Type / 유형 | Display / 표시 | Auto Time Calc / 시간 자동 계산 |

|------|------|---------------|

| WORK | Office (gray) / 출근 (회색) | Start + 9 hours / 시작시간 + 9시간 |

| REMOTE | Remote (green) / 재택 (초록) | Start + 9 hours / 시작시간 + 9시간 |

| DAY_OFF | Day Off (red) / 휴가 (빨강) | No time shown / 시간 표시 없음 |

| AM_HALF | AM Half (yellow) / 오전반차 (노랑) | 13:00–17:00 (PM only) / 오후만 근무 |

| PM_HALF | PM Half (orange) / 오후반차 (주황) | Start–12:00 (AM only) / 오전만 근무 |

| MENSTRUATION | Menstrual Leave (pink) / 생리휴가 (분홍) | No time shown / 시간 표시 없음 |

3.4 Key Features / 주요 기능

Register / Edit Schedule / 일정 등록·수정

  1. Click own row cell (only editable dates) / 본인 행의 셀 클릭
  2. Select work type and start time in bottom form / 하단 폼에서 근무유형과 시작시간 선택
  3. End time auto-calculated by type / 종료시간은 유형에 따라 자동 계산
  4. Click [Save / 저장] / 버튼 클릭

Delete Schedule / 일정 삭제

  • Select cell with existing schedule → Click [Delete / 삭제] / 기존 일정이 있는 셀 선택 → 삭제 버튼 클릭

Batch Registration / 주간 일괄 등록

  • [Batch Register / 일괄 등록] button for Mon–Fri in one step / 월~금 5일치를 한 번에 등록
  • Individual work type/start time per day / 각 요일별 근무유형·시작시간을 개별 설정 가능

Editable Range / 편집 가능 범위

  • Own row only editable (other members: view-only) / 본인 행만 수정 가능
  • Editable period: Next Monday to 1 month from today / 다음 주 월요일부터 1개월 후까지
  • Past weeks cannot be edited / 이미 지난 주의 일정은 수정 불가

Start Time Options / 시작시간 옵션

Selectable in 30-min intervals from 06:00 to 12:00 / 06:00 ~ 12:00 사이 30분 간격

28.1 커스텀 앱 Custom Apps

Feature for hosting external web applications in an iframe. / 외부 웹 애플리케이션을 iframe으로 호스팅하는 기능입니다.

| Auth Mode / 인증 모드 | Description / 설명 |

|-----------|------|

| api_key | Auto-pass API key (?api_key) / API 키 자동 전달 |

| none | No auth / 인증 없음 |

| Open Mode / 열기 모드 | Description / 설명 |

|-----------|------|

| iframe | Host in page via iframe / iframe으로 페이지 내 호스트 |

  • "Open in New Tab" fallback button when iframe load is blocked / iframe 로드 차단 시 대체 버튼 제공
  • Current language auto-passed as ?locale= parameter / 현재 언어가 파라미터로 자동 전달

28.2 다국어 지원 Multi-Language Support

  • Full UI supports Korean / English / Vietnamese (3 languages) / 전체 UI가 3개 언어 지원
  • Switch via language selector in header right / 헤더 우측의 언어 선택기로 전환
  • Auto-translation panel for notes, TODOs, notices / 노트, 할일, 공지사항 등에 자동 번역 패널 제공

28.3 법인 전환 Entity Switching

  • Switch affiliated entity via header entity selector / 헤더의 법인 선택기로 소속 법인 전환
  • Projects, Billing etc. have per-entity data isolation / 프로젝트, 거래처관리 등은 법인 단위로 데이터 격리

28.4 글로벌 검색 Global Search

  • Integrated search across entire system via header search bar / 헤더 중앙의 검색바로 전체 시스템 통합 검색

  • Open AI assistant modal via bottom-right floating button / 우하단 플로팅 버튼으로 AI 어시스턴트 모달 열기
  • Auto-displayed once after login / 로그인 후 자동으로 한 번 표시
