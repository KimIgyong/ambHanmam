# 요구사항 분석서: UI 통일 개선

> 날짜: 2026-04-04

---

## 1. 요약

페이지별로 아이콘 스타일·상단 여백·제목 위치가 제각각인 현황을 통일하고,
Admin에서 작성한 메뉴 팁이 사용자 우측 패널에 표시되지 않는 버그를 수정한다.

### 2-4. Today 로고 소실 원인

이전 세션에서 `TodayPage.tsx`의 CalendarCheck 직접 아이콘을 `PageTitle` 컴포넌트로 교체하였다.  
`PageTitle`의 아이콘은 `MenuGuideContext → matched?.icon → myMenus (API) → ICON_MAP` 경로로 표시된다.

**소실 원인 추정:**  
`myMenus` 응답에서 TODAY 메뉴의 `icon` 값이 `ICON_MAP`에 등록된 키와 다르거나 누락된 경우 아이콘이 표시되지 않는다.  
→ 백엔드 메뉴 설정의 `icon` 필드와 프론트 `ICON_MAP` 키의 일치 여부 확인 필요.

### 2-5. Admin 메뉴 팁 → 사용자 우측 패널 미수신 원인

**데이터 흐름:**
```
Admin 작성 → PUT /entity-settings/site-config/menu-tips/:menuCode?entity_id=xxx
사용자 조회 → GET /entity-settings/site-config/menu-tips/:menuCode (JWT entityId 자동 사용)
```

**백엔드 GET 로직:**
```typescript
const entityId = user.entityId;         // JWT에서 추출
if (!entityId) return { success: true, data: null };
return this.siteConfigService.getMenuTip(entityId, menuCode);
```

**Admin PUT 조건 (`useUpsertMenuTip`):**
```typescript
const entityId = useEntityStore((s) => s.currentEntity?.entityId);
if (!entityId) throw new Error('entity_id is required');
```

**문제:** Admin 레이아웃(`/admin/permissions`)에서 `useEntityStore().currentEntity`가 설정되지 않은 상태이면  
`useMenuTips()`는 `enabled: false`로 비활성화되고, `useUpsertMenuTip()`은 오류를 던져 팁이 저장되지 않는다.

→ MenuGuideTab에서 entity 선택이 선행되어야 하며, 현재 어떤 법인의 팁을 편집하는지 컨텍스트가 없다.

## 3. TO-BE 요구사항

### 3-1. 전체 아이콘 스타일 기준

```
- 배경 div 없음
- 아이콘 컬러: text-gray-400 (회색 80%)
- 아이콘 크기: h-6 w-6 (PageTitle 기준)
- 배경색 클래스(bg-*) 완전 제거
```

### 3-2. 페이지 헤더 통일 기준

```
- 컴포넌트: <PageTitle> 사용 통일
- h1 폰트: text-2xl font-bold text-gray-900 (PageTitle 기본값)
- 컨텐츠 영역 여백: px-6 py-6 통일
```

### 3-3. 특수 케이스 요구사항

| 페이지 | 요구사항 |
|--------|---------|
| Today | 기존 원래 로고(CalendarCheck 아이콘 박스) 복원 또는 PageTitle 아이콘 정상 표시 |
| My Task | PageTitle 정상 노출 확인 |
| Project | 탭메뉴 **위**에 `[아이콘] Project` |
| Calendar | PageTitle 적용, 폰트 크기 통일 (text-2xl) |
| Drive | 배경박스 제거, 아이콘 gray-400 |
| My Leave | 아이콘 추가 (PageTitle icon 정상 연결) |
| Expense Request | 배경박스 제거, 아이콘 gray-400 |
| Partner (Billing) | 탭메뉴 **아래** 컨텐츠 영역에 `[아이콘] Partner` |
| Asset | 배경박스 제거, 아이콘 gray-400 |
| Accounting | PageTitle 적용, 아이콘 gray-400 |
| HR | 배경박스 제거, 아이콘 gray-400, 탭메뉴 **아래** 컨텐츠 영역에 `[아이콘] HR` |

### 3-4. Admin 메뉴 팁 수정 요구사항

- MenuGuideTab 상단에 법인 선택 드롭다운 추가
- 선택된 법인의 `entityId`로 `useMenuTips()`, `useUpsertMenuTip()` 동작
- 또는: Admin 사용자의 경우 entity_id를 URL 파라미터로 전달하는 방식으로 개선
