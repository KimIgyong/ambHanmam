# 자금업무 - 카드 사용내역 기능명세서

## 1. 개요
전사 법인카드 사용 데이터를 조회하고, 전자결재(지출 전표 상신) 매칭 및 미상신 항목을 관리합니다.

## 2. 화면 구성도 (Wireframe)
```
┌──────────────────────────────────────────────────────────────────────┐
│  카드 사용내역                                                          │
├──────────────────────────────────────────────────────────────────────┤
│ [검색]                                                                 │
│  2024년  [1]...[12]월  회사: [▼]  사용자: [____]  □ 미상신만 조회    │
│  카드사: [▼]  카드번호: [____]                                         │
│                          [조회] [선택목록 조회] [Upload목록]           │
│                          [카드사용내역 파일등록] [청구서 파일등록]      │
├──────────────────────────────────────────────────────────────────────┤
│사용자│승인일시     │승인금액 │구분│가맹점명  │등록일  │상신일  │출금액 │
│──────┼─────────────┼─────────┼────┼──────────┼────────┼────────┼───── │
│김철수│2024-11-05   │ 55,000  │승인│스타벅스  │2024-11 │2024-11 │55,000│
│      │             │         │    │⚠ 미상신  │        │        │      │
└──────────────────────────────────────────────────────────────────────┘
```

## 3. 입력 필드
| 필드명 | 유형 | 설명 |
|--------|------|------|
| 연도 | Number | |
| 월 | Radio Group | |
| 회사 | Select | |
| 사용자 | Text Input | |
| 미상신만 조회 | Checkbox | |
| 카드사 | Select | |
| 카드번호 | Text Input | 끝 4자리 |

## 4. 출력 필드
| 컬럼명 | 설명 |
|--------|------|
| 사용자 | 카드 소지자 |
| 승인일시 | 결제 일시 |
| 승인금액 | 카드 승인 금액 |
| 구분 | 승인/취소 |
| 가맹점명 | 결제처 |
| 등록일 | 시스템 등록일 |
| 상신일 | 전표 결재 기안일 |
| 출금액 | 계좌 출금 금액 |
| 출금날짜 | 실출금 일자 |

## 5. 액션 버튼
- `[조회]`
- `[선택목록 조회]`
- `[Upload목록]`
- `[카드사용내역 파일등록]` : CSV/Excel 업로드
- `[청구서 파일등록]` : 청구서 파일 업로드
