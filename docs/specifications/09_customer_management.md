# 영업업무 - 고객사 관리 기능명세서

## 1. 개요 및 핵심 기능
영업 대상 고객사(법인) 정보를 조회, 등록, 관리합니다. 계약 현황과 연결하여 영업 상태를 파악합니다.

## 3. 입력 필드
| 필드명 | 유형 | 필수 | 설명 |
|--------|------|------|------|
| 고객명 | Text Input | | 고객 회사명 키워드 검색 |
| 사업자번호 | Text Input | | 사업자등록번호 검색 |
| 담당자 | Text Input | | 영업/기술 담당자명 |

## 4. 출력 필드
| 컬럼명 | 설명 |
|--------|------|
| 번호 | 일련번호 |
| 고객(회사)명 | 법인명 (클릭 시 고객사 상세 이동) |
| 대표자명 | 법인 대표자 |
| 계약명 | 연결된 계약 명칭 (클릭 시 계약 상세 이동) |

## 5. 액션 버튼
| 버튼 | 동작 |
|------|------|
| `[검색]` | 조건에 맞는 고객사 조회 |
| `[초기화]` | 필드 초기화 |
| 고객명 클릭 | 고객사 상세 화면 이동 |
| 계약명 클릭 | 계약 관리 상세 화면 이동 |
