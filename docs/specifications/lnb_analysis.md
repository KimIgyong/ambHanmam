# 좌측 사이드 메뉴(LNB) 기능분석서 및 화면구성도

## 개요
한마음 시스템(hm.hanmam.kr)의 좌측 사이드 메뉴(LNB)인 '나의 업무', '업무연락', '전자결재', '쪽지' 영역에 대한 화면별 입력값, 출력값, 주요 기능을 분석한 문서입니다.

---

## 1. 나의 업무 (`/Daily?command=listUser`)
* **핵심 기능:** 개인 및 팀원의 주간 업무 계획과 실적을 시각적으로 관리하고 공유하는 타임라인형 대시보드입니다.
* **입력값 (검색 조건/네비게이션):**
  * 사원선택 탭 (상단에 나열된 사원 이름 클릭)
  * 날짜 선택 (직접 연월일 입력 또는 `<<`, `<`, `>`, `>>` 이동 버튼)
* **출력값 (목록 컬럼/그리드):**
  * 요일/날짜
  * 업무 계획 내용
  * 업무 실적 내용
* **주요 버튼:**
  * 날짜 이동 버튼
  * '일정 수정/등록' 아이콘 (신규 등록/수정 팝업 호출)

## 2. 업무연락 (`/Call?command=list`)
* **핵심 기능:** 사내 업무 협조 및 공지 사항을 주고받는 게시판 형태의 커뮤니케이션 도구입니다.
* **하위 메뉴:**
  * **처리중:** 본인이 담당한 미완료 업무 필터링 조회
  * **담당:** 처리 상태와 무관한 본인 담당 전체 업무 조회
  * **관심:** 관심 항목으로 등록된 게시물 모아보기
* **입력값 (검색 조건):**
  * 제목
  * 수신자 / 작성자 (이름 단위)
  * 번호
  * 프로젝트명
  * 구분 (알림, 업무, 분석, 설계, 개발, 과제)
  * 기간 (작성일 / 완료요청일)
  * 상태 (처리중, 완료, 종료)
* **출력값 (목록 컬럼):**
  * 번호
  * 제목
  * 작성자
  * 담당자
  * 작성일
  * 완료요청일
* **주요 버튼:**
  * `[검색]`
  * `[새글]` (게시글 작성)

## 3. 전자결재 (`/Doc?command=list`)
* **핵심 기능:** 사내에서 기안된 일반결재, 지출/매출 보고 등의 문서를 통합 관리하고 승인 프로세스를 진행합니다.
* **하위 메뉴:**
  * **승인할 문서:** 본인에게 결재 권한이 넘어와 대기 중인 목록
  * **진행중 문서:** 최종 승인이 안 나고 프로세스를 진행 중인 기안 건
  * **미상신 내역 (`/CardSpend?command=listNew`):** 시스템에 연동된 법인카드 사용분 중 아직 결재(전표 상신)하지 않은 비용 목록
* **입력값 (검색 조건):**
  * 제목
  * 문서종류 (일반결재, 휴가신청, 지출보고, 매출보고, 지출결의서 등)
  * 기안자
  * 기안일 (기간 캘린더 입력)
* **출력값 (목록 컬럼 - 결재문서):**
  * 번호
  * 제목
  * 구분
  * 기안자
* **출력값 (목록 컬럼 - 미상신 내역):**
  * 사용자, 회사, 카드명, 승인일시, 금액, 가맹점명
* **주요 버튼:**
  * `[검색]`
  * `[결재 작성]` 아이콘 (서식 선택 팝업 이동)

## 4. 쪽지 (`/Notice?command=list`)
* **핵심 기능:** 사내 직원간 단문 텍스트 메시지를 주고받는 시스템입니다.
* **하위 메뉴:**
  * **받은쪽지함:** 수신된 메시지 조회
  * **보낸쪽지함:** 발신한 메시지 이력 조회
* **입력값 (검색 조건):**
  * 내용
  * 수신일/송신일 (기간 조회)
  * 송신자/수신자
* **출력값 (목록 컬럼):**
  * 내용
  * 보낸사람 / 받는사람
  * 수신/송신 일시
* **주요 버튼:**
  * `[검색]`
  * `[쪽지 쓰기]` 아이콘
