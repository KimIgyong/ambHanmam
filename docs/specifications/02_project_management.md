# 경영정보 - 프로젝트관리 기능명세서

## 1. 개요 및 핵심 기능
로그인된 사용자가 PL(Project Leader) 또는 담당자로 참여 중인 전체 프로젝트 현황을 조회 및 모니터링합니다.

## 3. 입력 필드 (Search & Input Form)
- 별도 검색 조건 없음 (로그인 계정 기준, 권한 필터링 자동 수행)

## 4. 출력 필드 (Grid Output)
| 컬럼명 | 설명 |
|--------|------|
| 상위프로젝트 | 대형 사업 묶음명 |
| 프로젝트명 | 세부 프로젝트명 |
| PL | 프로젝트 담당자 |
| 구분 | 개발/운영/기타 |
| 투입 | 인력 투입 시간 합계 |
| 지출 | 집행된 비용 총액 |
| 예산 | 배정 예산 총액 |
| 보기 | 상세 보기 이동 버튼 |
| 상태 | 대기/진행중/완료 뱃지 |
| 시작일 / 종료일 | 프로젝트 기간 |

## 5. 액션 버튼
| 버튼 | 동작 |
|------|------|
| `[상세]` | 프로젝트 상세 화면 이동 |
