# 업무지원 - 문서발송대장 기능명세서

## 1. 개요
대외로 발송한 공식 공문서(제안서, 계획서 등)의 이력을 프로젝트별로 관리합니다.

## 2. 화면 구성도 (Wireframe)
```
┌──────────────────────────────────────────────────────────────────────┐
│  문서발송대장                                                     [등록]│
├──────────────────────────────────────────────────────────────────────┤
│  프로젝트명: [🔍________________]  담당자: [____]                     │
├──────────────────────────────────────────────────────────────────────┤
│  ☐ │ 문서번호     │ 문서제목            │ 사업명         │ 수신처       │
│────┼──────────────┼─────────────────────┼────────────────┼──────────── │
│ ☐  │ SEND-2024-001│ ABC 사업 제안서      │ ABC 시스템 구축 │ (주)ABC 담당  │
│ ☐  │ SEND-2024-002│ XYZ 결과보고서      │ XYZ ERP        │ 담당 김부장   │
└──────────────────────────────────────────────────────────────────────┘
```

## 3. 입력 필드
| 필드명 | 유형 | 설명 |
|--------|------|------|
| 프로젝트명 | Search Input | 돋보기 팝업 |
| 담당자 | Text Input | |

## 4. 출력 필드
| 컬럼명 | 설명 |
|--------|------|
| 선택 | 체크박스 |
| 문서번호 | 공문 번호 체계 |
| 문서제목 | |
| 사업명 | 연결 프로젝트 |
| 수신처 | 발송 대상 |

## 5. 액션 버튼
- `[등록]` : 신규 발송 문서 등록
- 프로젝트 🔍 → 검색 팝업
