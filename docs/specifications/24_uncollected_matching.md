# 자금업무 - 미처리 수금 관리 기능명세서

## 1. 개요
세금계산서가 발행되었으나 통장 입금 내역과 매칭되지 않은 항목(미수금)을 관리하고 수금 상태를 추적합니다.

## 2. 화면 구성도 (Wireframe)
```
┌──────────────────────────────────────────────────────────────────────┐
│  미처리 수금 관리                                                        │
├──────────────────────────────────────────────────────────────────────┤
│ [검색]                                                                 │
│  2024년  [1]...[12]월  회사: [▼]  담당자: [____]                      │
│                                    [조회] [선택목록 조회] [가상입금 등록]│
├──────────────────────────────────────────────────────────────────────┤
│     ⚠ 입금 미매칭 계산서 내역                                          │
│ 번호│날짜    │매출명         │계정과목│계산서액    │실입금액   │구분│담당자│
│─────┼────────┼───────────────┼────────┼────────────┼───────────┼────┼─────│
│  1  │2024-10 │ XYZ 10월 용역 │매출액  │ 5,500,000  │           │ ⚠ │김철수│
└──────────────────────────────────────────────────────────────────────┘
```

## 3. 입력 필드
| 필드명 | 유형 | 설명 |
|--------|------|------|
| 연도 | Number | |
| 월 | Radio Group | |
| 회사 | Select | |
| 담당자 | Text Input | |

## 4. 출력 필드
| 컬럼명 | 설명 |
|--------|------|
| 번호 | |
| 날짜 | 계산서 발행 연월 |
| 매출명 | 청구 건 명칭 |
| 계정과목 | |
| 계산서액 | 청구 금액 |
| 실입금액 | 실수금된 금액 |
| 구분 | 매칭상태 뱃지 |
| 담당자 | |
| 근거 | 증빙 서류 링크 |

## 5. 액션 버튼
- `[조회]`
- `[선택목록 조회]`
- `[가상입금 등록]` : 분할 수금 등 수동 처리
