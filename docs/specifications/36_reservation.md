# 알림게시판 - 공용 예약 (회의실/차량) 기능명세서

## 1. 개요
회사 공용 자원(회의실, 법인차량)의 예약 현황을 캘린더로 확인하고 신규 예약을 등록합니다.

## 2. 화면 구성도 (Wireframe)
```
┌──────────────────────────────────────────────────────────────────────┐
│  자원 예약    [회의실 예약] [차량 예약]                                 │
├──────────────────────────────────────────────────────────────────────┤
│              [< 이전]   2024년 11월   [다음 >]   [오늘]              │
├──────┬──────┬──────┬──────┬──────┬──────┬──────┤
│ 일   │ 월   │ 화   │ 수   │ 목   │ 금   │ 토   │
├──────┼──────┼──────┼──────┼──────┼──────┼──────┤
│  3   │  4   │  5   │  6   │  7   │  8   │  9   │
│      │🔵회의실│      │🚗 홍길│      │      │      │
│      │ 홍길동│      │ 동 출장│      │      │      │
└──────┴──────┴──────┴──────┴──────┴──────┴──────┘
```

## 3. 입력 필드
| 필드명 | 유형 | 설명 |
|--------|------|------|
| 자원 구분 | Tab | 회의실 / 차량 |
| 연/월 | Navigation | |
| 날짜 | Click | 예약 등록 팝업 |

**예약 등록 팝업 내 입력 항목:**
| 필드명 | 유형 |
|--------|------|
| 예약자 | 자동(로그인 계정) |
| 자원 | Select (회의실 A/B, 차량 번호) |
| 시작시간 | DateTime |
| 종료시간 | DateTime |
| 목적/사유 | Text |

## 4. 출력 필드
| 영역 | 설명 |
|------|------|
| 월간 달력 | 날짜별 예약 현황 블록 표시 |
| 예약 뱃지 | 예약자명 + 자원 구분 색상 |

## 5. 액션 버튼
- `[< 이전]` / `[다음 >]` / `[오늘]`
- 빈 날짜 클릭 → 예약 등록 팝업
- 기존 예약 클릭 → 상세 조회 (본인만 수정/취소 가능)
