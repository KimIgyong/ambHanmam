# 업무지원 - 문서수신대장 기능명세서

## 1. 개요
대외로부터 수신된 공식 문서(공문, 계약서 등)의 이력을 프로젝트별로 관리합니다.

```
│  문서수신대장                                                     [등록]│
├──────────────────────────────────────────────────────────────────────┤
│  프로젝트명: [🔍________________]  담당자: [____]                     │
├──────────────────────────────────────────────────────────────────────┤
│  ☐ │ 문서번호     │ 문서제목            │ 사업명         │ 발송처       │
│────┼──────────────┼─────────────────────┼────────────────┼──────────── │
│ ☐  │ RECV-2024-001│ 착수 공문           │ ABC 시스템 구축 │ (주)ABC 발주처│
│ ☐  │ RECV-2024-002│ 계약서 원본         │ XYZ ERP        │ XYZ 계약팀   │
└──────────────────────────────────────────────────────────────────────┘
```
