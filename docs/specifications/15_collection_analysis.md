# 영업업무 - 수금분석 기능명세서

## 1. 개요
프로젝트/계약 단위 수금 현황 및 미수금 내역을 분석하여 자금 흐름을 모니터링합니다.

## 2. 화면 구성도 (Wireframe)
```
┌─────────────────────────────────────────────────────────────────┐
│  수금분석                                                          │
├─────────────────────────────────────────────────────────────────┤
│ [검색]                                                             │
│  프로젝트: [🔍____]  요청자: [____]  회사: [____▼]  계약명: [____]│
│  종류: [계산서▼]                                        [조회]     │
├─────────────────────────────────────────────────────────────────┤
│                                    [전체글보기] [선택항목 보기]     │
│ 번호 │ 날짜    │ 제목          │ 고객사    │ 요청자 │ 발행액       │
│─────┼─────────┼───────────────┼──────────┼────────┼──────────── │
│  1  │2024-11  │ ABC 11월 청구 │ (주)ABC  │ 홍길동 │ 1,100,000   │
│  ⚠  │[미수금]  │               │          │        │             │
└─────────────────────────────────────────────────────────────────┘
```

## 3. 입력 필드
| 필드명 | 유형 | 설명 |
|--------|------|------|
| 프로젝트 | Search Input | 프로젝트 검색 팝업 |
| 요청자 | Text Input | |
| 회사 | Select | 당사/계열사 구분 |
| 계약명 | Text Input | |
| 종류 | Select | 계산서, 카드, 미발행 |

## 4. 출력 필드
| 컬럼명 | 설명 |
|--------|------|
| 번호 | 일련번호 |
| 날짜 | 발행 요청 일자 |
| 제목 | 계산서 요청 제목 |
| 고객사 | 청구 대상 고객사 |
| 요청자 | 담당 영업자 |
| 발행액 | 청구/발행 금액 |

## 5. 액션 버튼
- `[조회]`
- `[전체글보기]`
- `[선택항목 보기]`
- 제목 클릭 → 상세 조회
- 고객사명 클릭 → 고객사 상세
