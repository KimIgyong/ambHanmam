# LNB - 쪽지 기능명세서

## 1. 개요
사내 임직원 간 단문 텍스트 메시지(쪽지)를 주고받는 1:1 메시지 시스템입니다.

## 2. 화면 구성도 (Wireframe)
```
┌──────────────────────────────────────────────────────────────────────┐
│  쪽지함        [받은쪽지함 🔴3] [보낸쪽지함]                    [쪽지쓰기]│
├──────────────────────────────────────────────────────────────────────┤
│ [검색]                                                                 │
│  내용: [________________]  보낸사람: [____]  수신일: [____]~[____]   │
│                                                           [검색]       │
├──────────────────────────────────────────────────────────────────────┤
│  발신자   │ 내용 미리보기                     │ 수신 일시              │
│───────────┼───────────────────────────────────┼──────────────────────│
│ 🔴 홍길동 │ 내일 회의 자료 검토 부탁드립니다...│ 2024-11-06 14:30      │
│  팀장     │ 이번주 KPI 정리 요청합니다...     │ 2024-11-05 09:00      │
└──────────────────────────────────────────────────────────────────────┘
```

## 3. 입력 필드
| 필드명 | 유형 | 설명 |
|--------|------|------|
| 내용 | Text Input | 키워드 검색 |
| 보낸사람/받은사람 | Text Input | |
| 수신/발신일 | Date Range | |

**쪽지 작성 팝업:**
| 필드명 | 유형 |
|--------|------|
| 받는사람 | 사원 선택 (자동완성) |
| 내용 | Textarea |

## 4. 출력 필드
| 컬럼명 | 설명 |
|--------|------|
| 발신자 | 보낸 사람 (미읽음 시 빨간 점 표시) |
| 내용 미리보기 | 첫 50자 미리보기 |
| 수신 일시 | |

## 5. 액션 버튼
- `[쪽지쓰기]` : 쪽지 작성 팝업
- `[받은쪽지함]` / `[보낸쪽지함]` 탭
- `[검색]`
- 쪽지 클릭 → 내용 전체 읽기 (읽음 처리)
