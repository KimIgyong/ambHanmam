# 업무지원 - 설계문서 조회 기능명세서

## 1. 개요
등록된 설계 문서를 다양한 조건(유형, 상태, 키워드)으로 목록 조회합니다.

## 2. 화면 구성도 (Wireframe)
```
┌──────────────────────────────────────────────────────────────────────┐
│  설계문서 조회                                                          │
├──────────────────────────────────────────────────────────────────────┤
│  설계서 종류: [화면설계서▼]  처리상태: [전체▼]  제목: [____________] │
│                                                            [검색]     │
├──────────────────────────────────────────────────────────────────────┤
│  번호 │ 제목                     │ 처리상태  │ 작성자   │ 수정일      │
│───────┼──────────────────────────┼──────────┼──────────┼───────────  │
│   1   │ 회의관리 화면 설계서      │ ● 완성   │ 김철수   │ 2024-10-20  │
│   2   │ 계약관리 화면 설계서      │ ◑ 개발중  │ 이영희   │ 2024-11-01  │
└──────────────────────────────────────────────────────────────────────┘
```

## 3. 입력 필드
| 필드명 | 유형 | 설명 |
|--------|------|------|
| 설계서 종류 | Select | |
| 처리상태 | Select | 전체/준비/설계중/개발중/검증중/완성 |
| 제목 | Text Input | 키워드 검색 |

## 4. 출력 필드
| 컬럼명 | 설명 |
|--------|------|
| 번호 | |
| 제목 | (클릭 시 상세) |
| 처리상태 | 상태 뱃지 |
| 작성자 | |
| 수정일 | 최종 수정 날짜 |
