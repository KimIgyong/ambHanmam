# 업무지원(Business Support) 기능분석서 및 화면구성도

## 개요
한마음 시스템(hm.hanmam.kr)의 '업무지원' 하위 메뉴에 대한 각 화면별 입력값(검색조건), 출력값(그리드/테이블 컬럼), 그리고 주요 액션 버튼을 분석한 문서입니다.

---

## 1. 슬기로운 회사생활 (`/Bbs?command=list&bbsMasterNo=53`)
* **핵심 기능:** 사내 공지사항 및 회사 생활 관련 정보를 공유하는 게시판입니다.
* **입력값 (검색 조건):**
  * 제목
  * 등록일 (시작 ~ 종료)
  * 등록자
* **출력값 (목록 컬럼):**
  * 번호
  * 제목
  * 등록일
  * 작성자
* **주요 버튼:**
  * `[검색]`
  * `[글 쓰기]`

## 2. 개발관리게시판 (`/Bbs?command=list&bbsMasterNo=88`)
* **핵심 기능:** 개발 일정 및 관련 자료를 공유하고 관리하는 게시판입니다.
* **입력값 (검색 조건):**
  * 제목
  * 등록일 (시작 ~ 종료)
  * 등록자
* **출력값 (목록 컬럼):**
  * 번호
  * 제목
  * 등록일
  * 작성자
* **주요 버튼:**
  * 일정 / 자료 탭 전환 버튼
  * `[검색]`

## 3. 회사자료 (`/Bbs?command=list&bbsMasterNo=50`)
* **핵심 기능:** 사업자 정보, 증명서 등 사내 공용 문서를 관리하는 게시판입니다.
* **입력값 (검색 조건):**
  * 제목
  * 등록일 (시작 ~ 종료)
  * 등록자
* **출력값 (목록 컬럼):**
  * 번호
  * 제목
  * 등록일
  * 작성자
* **주요 버튼:**
  * `[검색]`

## 4. 문서발송대장 (`/DocRegister?command=list&registerType=S`)
* **핵심 기능:** 대외로 발송한 공식 문서의 이력을 관리합니다.
* **입력값 (검색 조건):**
  * 프로젝트명 (검색 아이콘)
  * 담당자 (사원명)
* **출력값 (목록 컬럼):**
  * 선택 (체크박스)
  * 문서번호
  * 문서제목
  * 사업명
  * 수신처
* **주요 버튼:**
  * 프로젝트 검색 아이콘

## 5. 문서수신대장 (`/DocRegister?command=list&registerType=R`)
* **핵심 기능:** 대외로부터 수신된 공식 문서의 이력을 관리합니다.
* **입력값 (검색 조건):**
  * 프로젝트명 (검색 아이콘)
  * 담당자 (사원명)
* **출력값 (목록 컬럼):**
  * 선택 (체크박스)
  * 문서번호
  * 문서제목
  * 사업명
  * 발송처
* **주요 버튼:**
  * 프로젝트 검색 아이콘

## 6. 설계문서 관리 (`/Design?command=tree`)
* **핵심 기능:** 시스템 설계 항목별 진행 상태(준비, 설계, 개발, 검증, 완성)를 트리 구조로 관리합니다.
* **입력값 (검색 조건 / 액션):**
  * 설계서 종류 선택 (Select Box)
  * 트리 노드 선택
* **출력값 (현황판 / 트리):**
  * 진행 상태 현황 (등록/오류/설계중/개발중/검증중/완성 건수)
  * 기능별 트리 리스트
* **주요 버튼:**
  * 트리 확장/축소 (+/-)
  * 상태별 수치 클릭 시 대시보드 표출

## 7. 설계문서 조회 (`/Design?command=list`)
* **핵심 기능:** 등록된 설계 문서를 목록 형태로 조회합니다.
* **입력값 (검색 조건):**
  * 설계서 종류 선택
  * 처리상태 필터
  * 제목 키워드
* **출력값 (목록 컬럼):**
  * 번호
  * 제목
  * 처리상태
  * 작성자
* **주요 버튼:**
  * `[검색]`
