# 00. 메인 대시보드 기능명세서

## 1. 개요 및 핵심 기능
- **URL:** `https://hm.hanmam.kr/Main`
- 시스템 로그인 후 가장 먼저 노출되는 홈 화면으로, **공지사항, 개인 업무 현황, 팀 일정, 근태 상태, 임직원 기념일** 등을 한눈에 파악할 수 있는 통합 포털 대시보드입니다.
- 사이드바(LNB)의 업무연락/전자결재/쪽지 미처리 건수를 뱃지로 즉시 확인 가능합니다.

## 3. 영역별 상세 구성

### 3-1. GNB (Global Navigation Bar)
| 구성 요소 | 설명 |
|-----------|------|
| 회사 로고 | 클릭 시 메인 화면(`/Main`) 이동 |
| 사원명 검색 🔍 | 사원명 입력 후 조직도/사원 빠른 검색 |
| Video 📹 아이콘 | 화상회의 기능 (외부 연동) |
| Mail ✉ 아이콘 | 사내/외 이메일 연결 |
| 메인 메뉴 탭 | 경영정보, 영업업무, 서비스업무, 업무지원, 알림게시판 |

### 3-2. LNB (Left Navigation Bar - 개인 업무 사이드바)
| 항목 | 설명 |
|------|------|
| 프로필 정보 | 로그인 사용자명, 접속 상태(외부/사내), 최종로그인 |
| 나의 업무 🔴 | 미완료 업무 뱃지 표시 / 클릭 시 `/Daily` 이동 |
| 업무연락 | 처리중/담당/관심 3개 탭, 건수 뱃지 표시 |
| 전자결재 | 승인할문서/진행중문서/미상신내역 건수 뱃지 |
| 쪽지 | 받은/보낸 쪽지함, 미확인 건수 표시 |
| 나의 메뉴 | 사용자 커스텀 즐겨찾기 메뉴 등록 영역 |

### 3-3. 메인 위젯 영역 (Main Content)
| 위젯명 | 위치 | 설명 |
|--------|------|------|
| 📢 공지사항 | 상단 좌측 | 최신 전사 공지/사내 게시물 목록 5~10건 |
| 📝 최근 게시물 | 상단 우측 | 생일축하, 입사 인사 등 커뮤니티 게시물 |
| 📅 일정공유 | 중단 좌측 | 금일/이번주 등록된 공유 일정 요약 |
| 👥 근태현황 | 중단 우측 | 출근/외근/출장 실시간 인원 집계 |
| 🎂 기념일 | 하단 전체 | 이번주 생일/입사기념일 임직원 프로필 카드 |

## 4. 입력 필드
| 필드명 | 유형 | 설명 |
|--------|------|------|
| 사원명 검색 | Text Input | GNB 상단 검색창 |

## 5. 출력 필드
| 영역/위젯 | 출력 항목 |
|----------|---------|
| 공지사항 | 카테고리, 제목, 작성자, 등록일 |
| 최근 게시물 | 종류(생일/인사), 내용 미리보기, 날짜 |
| 일정공유 | 일정 제목, 날짜, 담당자 |
| 근태현황 | 출근/외근/출장 인원 수 |
| 기념일 | 사원 사진, 이름, 기념일 구분(생일/입사), 날짜 |
| LNB 뱃지 | 업무연락/결재/쪽지 미처리 건수 |

## 6. 액션 버튼
| 버튼 | 위치 | 동작 |
|------|------|------|
| 🔍 검색 | GNB 우측 | 사원 이름 조직도 검색 팝업 |
| `[+]` 더보기 | 위젯 우측 상단 | 해당 위젯의 전체 목록 페이지 이동 |
| `[✏]` 작성 | LNB 업무연락/결재/쪽지 옆 | 해당 기능의 새 글 작성 팝업 |
| 기념일 카드 클릭 | 기념일 위젯 | 해당 사원 프로필 팝업 |
| 일정 항목 클릭 | 일정공유 위젯 | 일정 상세 팝업 표시 |
