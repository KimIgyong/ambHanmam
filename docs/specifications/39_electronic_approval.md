# LNB - 전자결재 기능명세서

## 1. 개요
사내 에서 기안된 일반결재, 지출보고, 휴가신청 등 다양한 문서의 결재 프로세스를 관리합니다.

## 2. 화면 구성도 (Wireframe)
```
┌──────────────────────────────────────────────────────────────────────┐
│  전자결재   [승인할 문서 2] [진행중 문서] [미상신 내역]             [결재작성]│
├──────────────────────────────────────────────────────────────────────┤
│ [검색]                                                                 │
│  제목: [________]  문서종류: [전체▼]  기안자: [____]                 │
│  기안일: [2024-11-01]~[2024-11-30]                   [검색]           │
├──────────────────────────────────────────────────────────────────────┤
│ 번호 │ 제목                    │ 구분      │ 기안자  │ 상태          │
│──────┼─────────────────────────┼───────────┼─────────┼───────────── │
│  15  │ 11월 출장 지출보고서    │ 지출결의서 │ 김철수  │ 🟡 결재중    │
│  14  │ 연차 신청 (11/10~11/11) │ 휴가신청   │ 이영희  │ ✅ 승인완료  │
└──────────────────────────────────────────────────────────────────────┘
```

## 3. 입력 필드
| 필드명 | 유형 | 설명 |
|--------|------|------|
| 제목 | Text Input | |
| 문서종류 | Select | 일반결재, 휴가신청, 지출보고, 매출보고 등 |
| 기안자 | Text Input | |
| 기안일 | Date Range | |

## 4. 출력 필드
| 컬럼명 | 설명 |
|--------|------|
| 번호 | |
| 제목 | (클릭 시 결재 상세) |
| 구분 | 문서 유형 |
| 기안자 | |
| 상태 | 결재중/승인완료/반려 뱃지 |

## 5. 액션 버튼
- `[결재작성]` : 서식 선택 후 기안서 작성
- `[승인할 문서]` / `[진행중 문서]` / `[미상신 내역]` 탭
- `[검색]`
- 제목 클릭 → 결재 상세 (승인/반려 버튼 포함)
