# LNB - 업무연락 기능명세서

## 1. 개요
사내 업무 협조 요청 및 공지를 주고받는 메인 커뮤니케이션 도구입니다. 처리중/담당/관심 탭으로 구분됩니다.

## 2. 화면 구성도 (Wireframe)
```
┌──────────────────────────────────────────────────────────────────────┐
│  업무연락          [처리중 3] [담당] [관심]                       [새글]│
├──────────────────────────────────────────────────────────────────────┤
│ [검색]                                                                 │
│  제목: [________]  수신자: [____]  프로젝트: [____]  구분: [업무▼]   │
│  기간: [____]~[____]  상태: ○전체 ○처리중 ○완료                    │
│                                                           [검색]       │
├──────────────────────────────────────────────────────────────────────┤
│ 번호 │ 제목                    │ 작성자 │ 담당자 │ 작성일    │ 완료요청일│
│──────┼─────────────────────────┼────────┼────────┼───────────┼─────────│
│  25  │ 🔴 ABC 장애 처리 요청   │ 홍길동  │ 김철수  │ 2024-11-06│2024-11-06│
│  24  │ 주간보고 작성 요청       │ 팀장   │ 전체   │ 2024-11-04│2024-11-08│
└──────────────────────────────────────────────────────────────────────┘
```

## 3. 입력 필드
| 필드명 | 유형 | 설명 |
|--------|------|------|
| 제목 | Text Input | |
| 수신자/작성자 | Text Input | |
| 프로젝트 | Text Input | |
| 구분 | Select | 알림, 업무, 분석, 설계, 개발, 과제 |
| 기간 | Date Range | |
| 상태 | Radio | 전체/처리중/완료 |

## 4. 출력 필드
| 컬럼명 | 설명 |
|--------|------|
| 번호 | |
| 제목 | 긴급도 색상 표시 (클릭 시 상세) |
| 작성자 | 송신자 |
| 담당자 | 수신/처리 담당자 |
| 작성일 | |
| 완료요청일 | 처리 기한 |

## 5. 액션 버튼
- `[새글]` : 업무 연락 작성
- `[처리중]` / `[담당]` / `[관심]` 탭 전환
- `[검색]`
- 제목 클릭 → 상세/답변 화면
