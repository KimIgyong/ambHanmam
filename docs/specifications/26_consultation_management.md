# 서비스업무 - 상담관리 기능명세서

## 1. 개요
접수된 상담 내역을 목록으로 조회하고, 처리 상태를 관리합니다.

## 2. 화면 구성도 (Wireframe)
```
┌──────────────────────────────────────────────────────────────────────┐
│  상담관리                                                              │
├──────────────────────────────────────────────────────────────────────┤
│ [검색]                                                                 │
│  사용자: [____]  계약명: [🔍____]  내용: [____]                       │
│  처리상태: ○ 전체  ○ 접수  ○ 완료                                    │
│  접수일: [2024-11-01] ~ [2024-11-30]                                 │
│                                                  [🔍 검색] [초기화]   │
├──────────────────────────────────────────────────────────────────────┤
│ 번호 │ 사용자  │ 문의내용(요약)         │ 계약명     │ 접수시간 │ 접수자 │
│──────┼─────────┼────────────────────────┼────────────┼──────────┼───────│
│  1   │ ABC고객  │ 로그인이 안 됩니다...  │ ABC 유지보수│ 11-05 10 │홍길동  │
│      │  ✅ 완료│                        │            │           │       │
└──────────────────────────────────────────────────────────────────────┘
```

## 3. 입력 필드
| 필드명 | 유형 | 설명 |
|--------|------|------|
| 사용자 | Text Input | 문의자명 |
| 계약명 | Search | 돋보기 팝업 검색 |
| 문의내용 | Text Input | 키워드 검색 |
| 처리상태 | Radio | 전체/접수/완료 |
| 접수일 | Date Range | |

## 4. 출력 필드
| 컬럼명 | 설명 |
|--------|------|
| 번호 | |
| 사용자 | 문의 접수자 |
| 문의내용 | 내용 요약 (클릭 시 상세) |
| 계약명 | 연결 계약 |
| 접수시간 | 접수 일시 |
| 접수자 | 처리 담당자 |

## 5. 액션 버튼
- `[🔍 검색]`
- `[초기화]`
- 항목 클릭 → 상세 조회/처리 화면
