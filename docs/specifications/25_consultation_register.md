# 서비스업무 - 상담접수 기능명세서

## 1. 개요
고객의 서비스 요청(장애, 영업 문의, 일반 상담)을 시스템에 접수하는 입력 폼 화면입니다.

## 2. 화면 구성도 (Wireframe)
```
┌──────────────────────────────────────────────────────────────────────┐
│  상담접수 등록                                                          │
├──────────────────────────────────────────────────────────────────────┤
│  사용자:   [________________]  연락처: [___-____-____]               │
│  이메일:   [________________]                                         │
│  상담분류: ○ 일반상담  ○ 영업문의  ○ 장애문의                        │
│  계약명:   [🔍 계약 선택__________________]                           │
│  ─────────────────────────────────────────────────────────────────  │
│  문의내용: ┌─────────────────────────────────────────────────────┐   │
│            │                                                       │   │
│            │          (필수 입력 영역)                             │   │
│            └─────────────────────────────────────────────────────┘   │
│  첨부파일: [파일선택]                                                 │
│  연락시한: [2024-11-06] [14]시 [00]분                                │
│            [즉시] [10분 후] [1시간 후] [금일 중]                     │
│                                                          [등 록]      │
└──────────────────────────────────────────────────────────────────────┘
```

## 3. 입력 필드
| 필드명 | 유형 | 필수 | 설명 |
|--------|------|------|------|
| 사용자 | Text | O | 문의 접수자명 |
| 연락처 | Text | O | 전화번호 |
| 이메일 | Email | | 이메일 주소 |
| 상담분류 | Radio | O | 일반/영업/장애 |
| 계약명 | Search | | 연결 계약 (팝업 선택) |
| 문의내용 | Textarea | O | 상세 문의 내용 |
| 첨부파일 | File Upload | | |
| 연락시한 | DateTime | | 처리 요청 기한 |

## 4. 출력 필드
- 등록 완료 알림(Toast/Modal)만 표시, 별도 목록 출력 없음

## 5. 액션 버튼
- `[파일선택]` : 첨부파일 선택
- `[즉시]` / `[10분 후]` / `[1시간 후]` / `[금일 중]` : 시한 자동 설정
- `[등 록]` : 상담 접수 데이터 저장
