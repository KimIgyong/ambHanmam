# 경영정보 - 연간사업계획 기능명세서

## 1. 개요 및 핵심 기능
연간 단위로 기획된 주요 사업 및 부서별 목표/계획을 열람합니다. 권한이 없는 경우 담당 프로젝트 화면으로 리다이렉트됩니다.

## 2. 화면 구성도 (Wireframe)

```
┌──────────────────────────────────────────────────────────────────────────┐
│  🏢 한마음 시스템 > 경영정보 > 연간사업계획                                │
├──────────────────────────────────────────────────────────────────────────┤
│  [검색 조건]                                                               │
│  ┌──────────────────────────────────────────────────────────────────────┐ │
│  │  [<] 2024년 [>]                                         [검색]       │ │
│  └──────────────────────────────────────────────────────────────────────┘ │
├──────────────────────────────────────────────────────────────────────────┤
│  [데이터 그리드]                                                           │
│  ┌────────────────────────────┬──────────────┬──────────┬──────────────┐ │
│  │       사업계획명            │  관련 조직    │  담당자  │  연간 목표액  │ │
│  ├────────────────────────────┼──────────────┼──────────┼──────────────┤ │
│  │ 2024 솔루션 사업 확대 계획  │ 사업부 1팀   │ 홍길동   │   500,000,000 │ │
│  ├────────────────────────────┼──────────────┼──────────┼──────────────┤ │
│  │ 2024 클라우드 전환 계획     │ 기술지원팀   │ 김철수   │   200,000,000 │ │
│  └────────────────────────────┴──────────────┴──────────┴──────────────┘ │
│                                                                            │
│  ⚠️ 권한이 없는 경우 이 화면에서 [담당 프로젝트 관리] 화면으로 이동됩니다. │
└──────────────────────────────────────────────────────────────────────────┘
```

## 3. 입력 필드
| 필드명 | 유형 | 설명 |
|--------|------|------|
| 연도 | Number + 화살표버튼 | `[<]` `[>]` 로 이전/다음 연도 이동 |

## 4. 출력 필드
| 컬럼명 | 설명 |
|--------|------|
| 사업계획명 | 연간 사업 계획 명칭 |
| 관련 조직 | 담당 부서/팀 |
| 담당자 | 사업 책임자 |
| 연간 목표액 | 목표 매출/달성 수치 |

## 5. 액션 버튼
| 버튼 | 동작 |
|------|------|
| `[<]` / `[>]` | 이전/다음 연도 이동 |
| `[검색]` | 해당 연도 데이터 조회 |
