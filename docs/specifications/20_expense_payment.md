# 자금업무 - 경비지급 관리 기능명세서

## 1. 개요
임직원이 신청한 경비의 지급 여부를 확인하고, 실제 계좌 이체(송금) 처리를 진행합니다.

## 2. 화면 구성도 (Wireframe)
```
┌──────────────────────────────────────────────────────────────────────┐
│  경비지급 관리                                                          │
├──────────────────────────────────────────────────────────────────────┤
│ [검색]                                                                 │
│  2024년  [1][2][3]...[12]월  회사: [▼]  사용자: [____]  처리: [미처리▼]│
│                                                      [조회]            │
├──────────────────────────────────────────────────────────────────────┤
│                                                    [송금처리하기]        │
│ 날짜   │ 이름  │ 금액    │ 계정과목  │ 구분   │지급자│지급날짜│문서번호│결재│
│────────┼───────┼─────────┼──────────┼────────┼──────┼────────┼────────┼───│
│2024-11 │김철수 │ 80,000  │ 여비교통비│ 출장비 │ 관리자│2024-11│ D001  │완료│
└──────────────────────────────────────────────────────────────────────┘
```

## 3. 입력 필드
| 필드명 | 유형 | 설명 |
|--------|------|------|
| 연도 | Number | |
| 월 | Radio Group | |
| 회사 | Select | |
| 사용자 | Text Input | 신청자명 |
| 처리상태 | Select | 미처리, 완료 |

## 4. 출력 필드
| 컬럼명 | 설명 |
|--------|------|
| 날짜 | 지출 신청일 |
| 이름 | 신청자 |
| 금액 | 신청/지급 금액 |
| 계정과목 | 회계 분류 |
| 구분 | 세부 구분 |
| 지급자 | 처리 담당 관리자 |
| 지급날짜 | 실이체 완료일 |
| 문서번호 | 결재 문서 번호 |
| 결재상태 | 완료/대기 뱃지 |

## 5. 액션 버튼
- `[조회]`
- `[송금처리하기]` : 선택된 항목 일괄 송금 처리 상태 변경
