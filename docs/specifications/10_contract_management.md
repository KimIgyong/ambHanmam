# 영업업무 - 계약관리 기능명세서

## 1. 개요
고객사와 체결한 계약 및 프로젝트 계약 현황을 조회/관리하며, 매출 자동등록 기능을 지원합니다.

## 2. 화면 구성도 (Wireframe)
```
┌─────────────────────────────────────────────────────────────────────┐
│  계약관리                                                             │
├─────────────────────────────────────────────────────────────────────┤
│ [검색 조건]                                                           │
│  계약명: [________]  상태: [계약▼]  담당자: [________]               │
│  기간: [2024-01-01] ~ [2024-12-31]  매출등록: [자동▼]               │
│                                                      [검색] [초기화] │
├────────────────────────────────────── ──────────────────────────────┤
│                                            [매출 자동등록] [전체글보기]│
│  번호 │ 계약명 │ 시작일 │ 종료일 │ 최종매출일 │ 상태 │ 담당 │ 계약금 │ 매출액│
│ ─────┼────────┼────────┼────────┼────────────┼──────┼──────┼────────┼───────│
│   1  │ ABC유지│2024-01 │2024-12 │ 2024-11-01 │ 계약 │ 홍길동│ 12,000 │ 8,000 │
└─────────────────────────────────────────────────────────────────────┘
```

## 3. 입력 필드
| 필드명 | 유형 | 설명 |
|--------|------|------|
| 계약명 | Text Input | 계약명 키워드 검색 |
| 계약상태 | Select | 계약, 중지, 종료 |
| 매출등록 | Select | 자동, 수동 |
| 사업번호 | Text Input | |
| 계산서 발행 | Select | 자동, 수동 |
| 계산서 종류 | Select | |
| 담당자 | Text Input | |
| 기간 | Date Range | 시작일 ~ 종료일 |

## 4. 출력 필드
| 컬럼명 | 설명 |
|--------|------|
| 번호 | 일련번호 (클릭 시 수정) |
| 계약명 | 계약 명칭 (클릭 시 상세) |
| 시작일 | 계약 개시일 |
| 종료일 | 계약 만료일 |
| 최종매출일 | 최근 매출 등록일 |
| 상태 | 계약/중지/종료 뱃지 |
| 담당 | 영업 책임자 |
| 계약금 | 총 계약 금액 |
| 매출액 | 누적 발생 매출 |

## 5. 액션 버튼
- `[매출 자동등록]` : 조건에 맞는 계약에 대해 매출 일괄 자동 등록
- `[전체글보기]` : 필터 해제 전체 조회
- 계약명 클릭 → 계약 상세 조회
- 번호 클릭 → 계약 수정 화면
