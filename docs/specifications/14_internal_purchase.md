# 영업업무 - 사내매입목록 기능명세서

## 1. 개요
사내 타 부서나 프로젝트로부터 매입한 내역(비용 발생 관점)을 조회합니다.

## 2. 화면 구성도 (Wireframe)
```
┌──────────────────────────────────────────────────────────────────────┐
│  사내매입목록                                                           │
├──────────────────────────────────────────────────────────────────────┤
│ [검색]                                                                 │
│  연도: [2024▼]  월: [11▼]  프로젝트: [🔍____]  구분: [사내매입▼]     │
│                                                           [조회]       │
├──────────────────────────────────────────────────────────────────────┤
│  ☐ 번호 │ 날짜    │ 매입명        │ 매입프로젝트  │ 매출프로젝트  │ 금액  │
│ ────────┼─────────┼───────────────┼──────────────┼──────────────┼─────│
│  ☐  1  │2024-11  │ 박영희 80H    │ B프로젝트    │ A프로젝트    │ 400만│
└──────────────────────────────────────────────────────────────────────┘
```

## 3. 입력 필드
| 필드명 | 유형 | 설명 |
|--------|------|------|
| 연도 | Select | |
| 월 | Select | |
| 프로젝트 | Search Input | 돋보기 팝업 |
| 거래구분 | Select | 사내매출, 사내매입, 인건비 |

## 4. 출력 필드
| 컬럼명 | 설명 |
|--------|------|
| 선택 | 체크박스 |
| 번호 | 일련번호 |
| 날짜 | 거래 연월 |
| 매입명 | 거래 명칭 |
| 매입프로젝트 | 비용 발생 프로젝트 |
| 매출프로젝트 | 수익 귀속 프로젝트 |
| 금액 | 정산 금액 |

## 5. 액션 버튼
- `[조회]`
- 매입명 클릭 → 상세 조회
- 번호 클릭 → 수정
