# 영업업무 - 사내매출목록 기능명세서

## 1. 개요
부서 또는 사내 프로젝트 간 발생하는 내부 매출 내역(사내 거래)을 조회하고 관리합니다.

## 2. 화면 구성도 (Wireframe)
```
┌──────────────────────────────────────────────────────────────────────┐
│  사내매출목록                                                           │
├──────────────────────────────────────────────────────────────────────┤
│ [검색]                                                                 │
│  연도: [2024▼]  월: [11▼]  프로젝트: [🔍____]  구분: [사내매출▼]     │
│                                                           [조회]       │
├──────────────────────────────────────────────────────────────────────┤
│  ☐ 번호 │ 날짜    │ 매출명        │ 매입프로젝트  │ 매출프로젝트  │ 금액  │
│ ────────┼─────────┼───────────────┼──────────────┼──────────────┼─────│
│  ☐  1  │2024-11  │ 김철수 120H   │ A프로젝트    │ B프로젝트    │ 600만│
└──────────────────────────────────────────────────────────────────────┘
```

## 3. 입력 필드
| 필드명 | 유형 | 설명 |
|--------|------|------|
| 연도 | Select | |
| 월 | Select | |
| 프로젝트 | Search Input | |
| 거래구분 | Select | 사내매출, 사내매입, 인건비 |

## 4. 출력 필드
| 컬럼명 | 설명 |
|--------|------|
| 선택 | 체크박스 |
| 번호 | 일련번호 (클릭 시 수정) |
| 날짜 | 거래 발생 연월 |
| 매출명 | 거래 명칭 (클릭 시 상세) |
| 매입프로젝트 | 비용 발생 프로젝트 |
| 매출프로젝트 | 수익 귀속 프로젝트 |
| 금액 | 거래 금액 |

## 5. 액션 버튼
- `[조회]` : 검색 실행
- 매출명 클릭 → 상세 조회
- 번호 클릭 → 수정 화면
