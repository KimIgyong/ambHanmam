# 자금업무(Finance / Funds Operations) 기능분석서 및 화면구성도

## 개요
한마음 시스템(hm.hanmam.kr)의 '자금업무' 하위 메뉴에 대한 각 화면별 입력값(검색조건), 출력값(그리드/테이블 컬럼), 그리고 주요 액션 버튼을 분석한 문서입니다. 매출 계산서 발행부터 경비 지급, 자금 흐름 모니터링까지 담당합니다.

---

## 1. 계산서 발행 업무 (`/Invoice?command=listPublish`)
* **기능:** 프로젝트별 매출에 대한 세금계산서 발행 요청 내역과 발행 상태를 관리합니다.
* **입력값 (검색 조건):**
  * 연/월 (라디오 버튼)
  * 프로젝트명
  * 요청자명
  * 회사명 (드롭다운)
  * 종류 (계산서/카드/미발행/가발행)
  * 상태 (취소/요청/발행)
  * 요청제목
* **출력값 (목록 컬럼):**
  * 번호
  * 날짜
  * 제목
  * 고객사
  * 요청자
  * 발행액
  * 수금액
  * 종류
  * 상태
* **주요 버튼:**
  * `[검색]`
  * `[선택목록 조회]`
  * `[Excel 출력]`

## 2. 경비지급 관리 (`/Expense?command=listPay`)
* **기능:** 임직원이 신청한 경비(여비교통비, 복리후생비 등)의 지급 여부를 확인하고 실제 송금 처리를 진행합니다.
* **입력값 (검색 조건):**
  * 연/월 (라디오 버튼)
  * 회사명
  * 사용자명
  * 처리상태 (미처리/완료)
* **출력값 (목록 컬럼):**
  * 날짜
  * 이름
  * 금액
  * 계정과목
  * 구분
  * 지급자
  * 지급날짜
  * 문서번호
  * 결재상태
* **주요 버튼:**
  * `[검색]`
  * `[송금처리하기]`

## 3. 카드 사용내역 (`/CardSpend?command=list`)
* **기능:** 전사 법인카드 사용 데이터를 조회하고, 비용 전표(상신) 매칭 여부를 관리합니다.
* **입력값 (검색 조건):**
  * 연/월 (라디오 버튼)
  * 회사구분
  * 사용자명
  * 미상신 내역 (체크)
  * 카드사 구분
  * 카드번호
* **출력값 (목록 컬럼):**
  * 사용자
  * 승인일시
  * 승인금액
  * 구분 (승인/취소)
  * 가맹점명
  * 등록일
  * 상신일
  * 출금액
  * 출금날짜
* **주요 버튼:**
  * `[검색]`
  * `[선택목록 조회]`
  * `[Upload목록]`
  * `[카드사용내역 파일등록]`
  * `[청구서 파일등록]`

## 4. 통장 입출금내역 (`/BankInout?command=list`)
* **기능:** 등록된 회사 통장의 실제 거래 내역(입/출금)을 기간별로 모니터링합니다.
* **입력값 (검색 조건):**
  * 회사 선택
  * 통장 선택
  * 조회 기간 (시작일 ~ 종료일)
* **출력값 (목록 컬럼):**
  * 거래일시
  * 입금액
  * 출금액
  * 잔액
  * 차액
  * 내용
  * 적요
* **주요 버튼:**
  * `[검색]`
  * `[출금보고 작성]`
  * `[입금보고 작성]`
  * `[선택목록 조회]`
  * `[통장거래내역 파일등록]`

## 5. 미처리 지출 관리 (`/Expense?command=listNoLink`)
* **기능:** 출금은 완료되었으나, 아직 정산 시스템상의 지출 내역과 연결되지 않은 항목을 관리합니다.
* **입력값 (검색 조건):**
  * 연/월 (라디오 버튼)
  * 회사명
  * 담당자명
* **출력값 (목록 컬럼):**
  * 번호
  * 날짜
  * 지출명
  * 계정과목
  * 금액
  * 출금액
  * 구분
  * 담당자
  * 근거
* **주요 버튼:**
  * `[검색]`
  * `[선택목록 조회]`
  * `[가상지출 등록]`

## 6. 미처리 수금 관리 (`/Invoice?command=listNoLink`)
* **기능:** 세금계산서가 발행되었으나, 통장 입금 내역과 매칭되지 않은 항목을 관리합니다.
* **입력값 (검색 조건):**
  * 연/월 (라디오 버튼)
  * 회사명
  * 담당자명
* **출력값 (목록 컬럼):**
  * 번호
  * 날짜
  * 지출명(매출명)
  * 계정과목
  * 금액
  * 입금액
  * 구분
  * 담당자
  * 근거
* **주요 버튼:**
  * `[검색]`
  * `[선택목록 조회]`
  * `[가상입금 등록]`
