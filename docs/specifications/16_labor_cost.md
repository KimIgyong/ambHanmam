# 영업업무 - 인건비관리 기능명세서

## 1. 개요
프로젝트에 투입된 인원별 인건비 내역(투입 시간 기반)을 조회하여 사내 코스트 현황을 파악합니다.

## 2. 화면 구성도 (Wireframe)
```
┌──────────────────────────────────────────────────────────────────────┐
│  인건비관리                                                             │
├──────────────────────────────────────────────────────────────────────┤
│ [검색]                                                                 │
│  연도: [2024▼]  월: [11▼]  프로젝트: [🔍____]  구분: [인건비▼]       │
│                                                           [조회]       │
├──────────────────────────────────────────────────────────────────────┤
│  ☐ 번호 │ 날짜    │ 대상자(매출명)    │ 매입프로젝트  │ 매출프로젝트  │
│ ────────┼─────────┼──────────────────┼──────────────┼──────────────│
│  ☐  1  │2024-11  │ 김철수 (160H)     │ A프로젝트    │ B프로젝트    │
│  ☐  2  │2024-11  │ 이영희 (80H)      │ B프로젝트    │ C프로젝트    │
└──────────────────────────────────────────────────────────────────────┘
```

## 3. 입력 필드
| 필드명 | 유형 | 설명 |
|--------|------|------|
| 연도 | Select | |
| 월 | Select | |
| 프로젝트 | Search Input | |
| 거래구분 | Select | 인건비 |

## 4. 출력 필드
| 컬럼명 | 설명 |
|--------|------|
| 선택 | 체크박스 |
| 번호 | 일련번호 |
| 날짜 | 발생 연월 |
| 대상자(매출명) | 이름 + 투입시간 |
| 매입프로젝트 | 비용 발생 PJT |
| 매출프로젝트 | 수익 귀속 PJT |

## 5. 액션 버튼
- `[조회]`
- 매출명 클릭 → 상세 조회
- 프로젝트명 클릭 → 프로젝트 상세
