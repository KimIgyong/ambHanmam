# 경영정보 - 지출예산/실적 관리 기능명세서

## 1. 개요 및 핵심 기능
편성된 예산 대비 실집행 지출액을 프로젝트별/계정과목별로 비교하여, 잔여 예산을 추적 관리합니다.

## 2. 화면 구성도 (Wireframe)

```
┌──────────────────────────────────────────────────────────────────────────┐
│  🏢 한마음 시스템 > 경영정보 > 지출예산/실적 관리                          │
├──────────────────────────────────────────────────────────────────────────┤
│  [검색 조건]                                                               │
│  ┌──────────────────────────────────────────────────────────────────────┐ │
│  │  [<] 2024년 [>]    ○1월 ○2월 ... ●11월 ○12월                       │ │
│  │                                              [검색] [🔄]             │ │
│  └──────────────────────────────────────────────────────────────────────┘ │
├──────────────────────────────────────────────────────────────────────────┤
│  [탭 전환]  ● 프로젝트별   ○ 계정항목별                                   │
├──────────────────────────────────────────────────────────────────────────┤
│  [프로젝트별 보기]                                                         │
│  ┌──────────────────────┬────────────┬────────────┬────────────┬────────┐ │
│  │     프로젝트명        │   예산액   │   지출액   │  잔여예산  │잔여누계│ │
```

## 3. 입력 필드
| 필드명 | 유형 | 설명 |
|--------|------|------|
| 연도 | Number + 화살표 | `[<]` `[>]` 이동 |
| 월 | Radio Group | 1~12월 선택 |

## 4. 출력 필드
| 컬럼명 | 설명 |
|--------|------|
| 프로젝트명 / 계정과목명 | 분석 단위 (탭 전환에 따라 변경) |
| 예산액 | 편성된 총 예산 금액 |
| 지출액 | 실집행된 금액 (진행바 표시) |
| 잔여예산 | 남은 예산 금액 |
| 잔여 누계 | 연간 누적 잔여 예산 |

## 5. 액션 버튼
| 버튼 | 동작 |
|------|------|
| `[검색]` | 선택 조건 기준 조회 |
| `[🔄]` | 데이터 새로고침 |
| `[프로젝트별]` 탭 | 프로젝트 단위 보기 |
| `[계정항목별]` 탭 | 계정과목 단위 보기 |
