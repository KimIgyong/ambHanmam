# 경영정보 - 수금현황 기능명세서

## 1. 개요 및 핵심 기능
프로젝트/고객사 단위 계약에서 계산서가 미발행되었거나 현금 수금이 미완료된 잔고를 추적하고 모니터링합니다.

## 3. 입력 필드
| 필드명 | 유형 | 설명 |
|--------|------|------|
| 회사 | Select | 고객사 선택 |
| 프로젝트 | Text Input | 프로젝트명 검색 |
| 담당자 | Text Input | 영업 담당자명 |
| 미발행 금액 | Select | 있음/없음/전체 |
| 미수금 금액 | Select | 있음/없음/전체 |
| 상태 | Checkbox | 계약, 중지, 종료 복수 선택 |
| 미수금 제외 기간 | Text Input | 개월 단위 입력 |

## 4. 출력 필드
| 컬럼명 | 설명 |
|--------|------|
| 선택 | 체크박스 |
| 번호 | 일련번호 |
| 계약명 | 계약 명칭 |
| 담당자 | 영업 책임자 |
| 매출액 | 총 계약 매출 금액 |
| 미발행액 | 계산서 미발행 금액 (경고 표시) |
| 미수금 | 실수금 되지 않은 잔액 (경고 표시) |
| 상태 | 계약/중지/종료 뱃지 |

## 5. 액션 버튼
| 버튼 | 동작 |
|------|------|
| `[검색]` | 조건에 맞는 수금 현황 조회 |
| `[전체글보기]` | 필터 해제 전체 목록 조회 |
| `[선택항목 보기]` | 체크된 항목만 필터링 |
