# 영업업무 - 사외매출목록 기능명세서

## 1. 개요
외부 고객사로부터 발생하는 매출 내역을 월별/프로젝트별로 조회합니다.

## 2. 화면 구성도 (Wireframe)
```
┌──────────────────────────────────────────────────────────────┐
│  사외매출목록                                                   │
├──────────────────────────────────────────────────────────────┤
│ [검색]                                                         │
│  연도: [2024▼]  월: [11▼]  프로젝트: [🔍____]  담당자: [____] │
│                                                     [조회]    │
├──────────────────────────────────────────────────────────────┤
│  ☐ 번호 │ 날짜     │ 매출명           │ 매출액     │ 발행액    │
│ ────────┼──────────┼──────────────────┼────────────┼──────── │
│  ☐  1  │ 2024-11  │ ABC 유지보수 매출 │ 2,200,000  │ 1,100,000│
│  ☐  2  │ 2024-11  │ XYZ 개발 용역    │ 5,000,000  │ 5,000,000│
└──────────────────────────────────────────────────────────────┘
```

## 3. 입력 필드
| 필드명 | 유형 | 설명 |
|--------|------|------|
| 연도 | Select | 조회 연도 |
| 월 | Select | 1~12월 |
| 프로젝트 | Search Input | 프로젝트 명 (돋보기 팝업) |
| 담당자 | Text Input | 영업 담당자명 |

## 4. 출력 필드
| 컬럼명 | 설명 |
|--------|------|
| 선택 | 체크박스 (다중 선택) |
| 번호 | 매출 번호 (클릭 시 수정) |
| 날짜 | 매출 발생 연월 |
| 매출명 | 매출 건 명칭 (클릭 시 상세) |
| 매출액 | 총 매출 계약금액 |
| 발행액 | 실발행(계산서 발행) 금액 |

## 5. 액션 버튼
- `[조회]` : 검색 실행
- 프로젝트 🔍 → 프로젝트 검색 팝업
- 매출명 클릭 → 상세 조회
- 번호 클릭 → 매출 수정
