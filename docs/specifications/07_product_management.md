# 경영정보 - 상품관리 기능명세서

## 1. 개요 및 핵심 기능
회사가 영업하거나 제공하는 서비스/솔루션 등 자사 마스터 상품군 목록을 조회하고 관리합니다.

## 2. 화면 구성도 (Wireframe)

```
┌──────────────────────────────────────────────────────────────────────────┐
│  🏢 한마음 시스템 > 경영정보 > 상품관리                                    │
├──────────────────────────────────────────────────────────────────────────┤
│                                                               [+ 등록]   │
├──────────────────────────────────────────────────────────────────────────┤
│  [데이터 그리드]                                                           │
│  ┌────┬───────────────────────────┬──────────────┬──────────┬────────┬───────┐│
│  │번호│         상품명             │  사업부 이익  │  단축명  │ 등록자 │ 등록일 ││
│  ├────┼───────────────────────────┼──────────────┼──────────┼────────┼───────┤│
│  │  1 │ BODA VOC                  │  VOC 사업부  │ BODA-V   │ 홍길동 │2023-01││
│  ├────┼───────────────────────────┼──────────────┼──────────┼────────┼───────┤│
│  │  2 │ BODA 솔루션               │  솔루션 사업부│ BODA-S   │ 김철수 │2022-06││
│  ├────┼───────────────────────────┼──────────────┼──────────┼────────┼───────┤│
│  │  3 │ 한마음 ERP                │  ERP 사업부  │ HM-ERP   │ 이영희 │2021-03││
│  └────┴───────────────────────────┴──────────────┴──────────┴────────┴───────┘│
│                           [이전] 1 2 [다음]                               │
└──────────────────────────────────────────────────────────────────────────┘
```

## 3. 입력 필드
- 별도 검색 조건 없음 (전체 등록 상품 자동 전체 조회)

## 4. 출력 필드
| 컬럼명 | 설명 |
|--------|------|
| 번호 | 일련번호 |
| 상품명 | 제품/서비스 명칭 (클릭 시 상세/수정) |
| 사업부 이익 | 귀속 사업부 정보 |
| 단축명 | 내부 약어/코드 |
| 등록자 | 최초 등록 담당자 |
| 등록일 | 시스템 등록일 |

## 5. 액션 버튼
| 버튼 | 동작 |
|------|------|
| `[+ 등록]` | 신규 상품 등록 폼 이동 |
| 상품명 클릭 | 상품 상세 조회 / 수정 |
