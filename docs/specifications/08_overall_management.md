# 경영정보 - 총괄경영정보 기능명세서

## 1. 개요 및 핵심 기능
근태 현황, 업무 협업 지표, 현금 자산, 매출 동향 등을 복합적으로 모아 임원/관리자가 한 화면에서 파악하는 통합 대시보드입니다.

## 2. 화면 구성도 (Wireframe)

```
┌──────────────────────────────────────────────────────────────────────────┐
│  🏢 한마음 시스템 > 경영정보 > 총괄경영정보                                │
├──────────────────────────────────────────────────────────────────────────┤
│  [검색 조건]                                                               │
│  [<] 2024년 [>]   ○1월 ○2월 ... ●11월 ○12월           [검색]           │
├──────────────────────────────────────────────────────────────────────────┤
│                                                                            │
│  ┌──────────────────────┐  ┌──────────────────────────────────────────┐   │
│  │  👥 근태현황          │  │  📋 업무연락 현황                         │   │
│  │  정상출근:  23명      │  │  처리중: 15건   완료: 82건                │   │
│  │  연차/외출:  3명      │  │  금일마감: 3건 ⚠️                        │   │
```

## 3. 입력 필드
| 필드명 | 유형 | 설명 |
|--------|------|------|
| 연도 | Number + 화살표 | |
| 월 | Radio Group | 1~12월 |

## 4. 출력 필드 (대시보드 위젯)
| 위젯명 | 표시 내용 |
|--------|---------|
| 근태현황 | 정상출근/연차/지각 인원 수 |
| 업무연락 현황 | 처리중/완료/기한 임박 건수 |
| 현금 자산 | 계좌별 잔액 및 총 자산 |
| 월별 매출 추이 | 최근 N개월 막대/선 차트 |

## 5. 액션 버튼
| 버튼 | 동작 |
|------|------|
| `[<]` / `[>]` | 연도 이동 |
| `[검색]` | 선택 월 기준 대시보드 갱신 |
| 위젯 클릭 | 해당 상세 화면으로 이동 |
