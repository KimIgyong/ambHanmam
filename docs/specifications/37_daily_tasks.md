# LNB - 나의 업무 기능명세서

## 1. 개요
개인 및 팀원의 주간/일별 업무 계획과 실적을 타임라인 형태로 관리하고 공유합니다.

## 2. 화면 구성도 (Wireframe)
```
┌──────────────────────────────────────────────────────────────────────┐
│  나의 업무                                                             │
├──────────────────────────────────────────────────────────────────────┤
│  [홍길동] [김철수] [이영희] [박지수]  ...  (부서원 탭 목록)            │
├──────────────────────────────────────────────────────────────────────┤
│                  [<< 처음] [< 이전]  2024-11-06 (수)  [다음 >] [끝 >>]│
├──────────────────────────────────────────────────────────────────────┤
│  날짜       │ 계획 내용                   │ 실적 내용                  │
│─────────────┼─────────────────────────────┼────────────────────────── │
│ 11/06 (수) │ ○ ABC 화면 설계 완료        │ ✓ ABC 화면 설계 완료       │
│             │ ○ 주간 보고서 작성          │ - 주간 보고서 작성 중      │
│─────────────┼─────────────────────────────┼────────────────────────── │
│ 11/07 (목) │ ○ API 개발 시작             │                            │
└──────────────────────────────────────────────────────────────────────┘
```

## 3. 입력 필드
| 필드명 | 유형 | 설명 |
|--------|------|------|
| 사원 선택 | Tab/Button | 상단 사원 이름 클릭으로 전환 |
| 날짜 | Navigation | 이전/다음 이동 |

## 4. 출력 필드
| 컬럼명 | 설명 |
|--------|------|
| 날짜 | 요일 포함 표시 |
| 계획 내용 | 해당 날짜 업무 계획 |
| 실적 내용 | 실제 완료/진행 실적 |

## 5. 액션 버튼
- 날짜 이동 `[<<]` `[<]` `[>]` `[>>]`
- 일정 항목 클릭/더블클릭 → 내용 등록/수정 팝업
