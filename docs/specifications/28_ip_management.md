# 서비스업무 - IP관리 기능명세서

## 1. 개요
사내 네트워크 IP 대역별 할당 현황과 사용자, 사용 목적을 관리합니다.

## 2. 화면 구성도 (Wireframe)
```
┌──────────────────────────────────────────────────────────────────────┐
│  IP 관리                                                               │
├──────────────────────────────────────────────────────────────────────┤
│  IP 대역: [192.168.1.x ▼]                                            │
├──────────────────────────────────────────────────────────────────────┤
│  IP 주소 ▲       │ 사용자     │ 목적              │ 할당일   │ 관리자  │
│──────────────────┼────────────┼───────────────────┼──────────┼────── │
│  192.168.1.1     │ 서버(공용) │ 게이트웨이         │ 2020-01  │ 관리자 │
│  192.168.1.10    │ 김철수      │ 개발 PC           │ 2022-03  │ 관리자 │
│  192.168.1.20    │ 이영희      │ 업무 노트북        │ 2023-06  │ 관리자 │
│  192.168.1.100   │ (미할당)    │                   │          │       │
└──────────────────────────────────────────────────────────────────────┘
```

## 3. 입력 필드
| 필드명 | 유형 | 설명 |
|--------|------|------|
| IP 대역 | Select | 등록된 내부망 서브넷 선택 |

## 4. 출력 필드
| 컬럼명 | 설명 |
|--------|------|
| IP 주소 | 할당된 IP (정렬 가능) |
| 사용자 | 사용 주체 |
| 목적 | 용도 설명 |
| 할당일 | 발급 날짜 |
| 관리자 | 발급 처리자 |

## 5. 액션 버튼
- 컬럼 헤더 클릭 → 정렬 (오름차순/내림차순)
- `[신규 할당]` : 미할당 IP에 사용자 배정
