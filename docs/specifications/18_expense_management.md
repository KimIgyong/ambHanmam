# 영업업무 - 지출관리 기능명세서

## 1. 개요
실발생 지출 내역(여비교통비, 접대비 등)을 증빙 기반으로 조회하고 관리합니다.

## 2. 화면 구성도 (Wireframe)
```
┌──────────────────────────────────────────────────────────────────────┐
│  지출관리                                                              │
├──────────────────────────────────────────────────────────────────────┤
│ [검색]                                                                 │
│  연도: [2024▼]  월: [11▼]  프로젝트: [🔍____]  계정과목: [여비교통▼] │
│                                               [조회]  [엑셀 다운로드] │
├──────────────────────────────────────────────────────────────────────┤
│  ☐ 번호 │ 날짜    │ 지출명       │ 프로젝트명  │ 계정과목  │ 금액  │ 부가세 │
│ ────────┼─────────┼──────────────┼─────────────┼──────────┼───────┼────── │
│  ☐  1  │2024-11  │ KTX 출장     │ A프로젝트   │ 여비교통비│ 80,000│  0   │
│  ☐  2  │2024-11  │ 팀 회식      │ B프로젝트   │ 복리후생비│150,000│  0   │
└──────────────────────────────────────────────────────────────────────┘
```

## 3. 입력 필드
| 필드명 | 유형 | 설명 |
|--------|------|------|
| 연도 | Select | |
| 월 | Select | |
| 프로젝트 | Search Input | |
| 계정과목 | Select | 여비교통비, 접대비, 복리후생비 등 |

## 4. 출력 필드
| 컬럼명 | 설명 |
|--------|------|
| 선택 | 체크박스 |
| 번호 | 일련번호 |
| 날짜 | 지출 발생일 |
| 지출명 | 지출 내용 |
| 프로젝트명 | 귀속 프로젝트 |
| 계정과목 | 회계 분류 |
| 금액 | 지출 원금 |
| 부가세 | VAT |

## 5. 액션 버튼
- `[조회]`
- `[엑셀 다운로드]`
- 지출명 클릭 → 상세 조회
- 번호 클릭 → 수정
