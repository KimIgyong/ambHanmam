# 영업업무 - 계산서 발행요청 기능명세서

## 1. 개요
프로젝트 진행에 따른 세금계산서, 카드 결제 등 발행 요청 내역을 생성하고 관리합니다.

## 2. 화면 구성도 (Wireframe)
```
┌──────────────────────────────────────────────────────────────────────┐
│  계산서 발행요청                                                        │
├──────────────────────────────────────────────────────────────────────┤
│ [검색]                                                                 │
│  2024년  [1월][2월][3월][4월][5월][6월][7월][8월][9월][10월][11월][12월]│
│  프로젝트: [________]  요청자: [______]  회사: [______▼]              │
│  종류: [계산서▼]  상태: [요청▼]  제목: [________]                     │
│                                                        [조회]         │
├──────────────────────────────────────────────────────────────────────┤
│                                               [엑셀 다운로드]          │
│  번호 │ 날짜     │ 제목             │ 고객사    │ 요청자 │ 발행액       │
│ ─────┼──────────┼──────────────────┼──────────┼────────┼─────────── │
│   1  │ 2024-11  │ ABC 11월 유지보수 │ (주)ABC  │ 홍길동 │ 1,100,000   │
└──────────────────────────────────────────────────────────────────────┘
```

## 3. 입력 필드
| 필드명 | 유형 | 설명 |
|--------|------|------|
| 연도 | Number/Select | 조회 연도 |
| 월 | Radio Group | 1~12월 |
| 프로젝트 | Text Input | 프로젝트 명칭 |
| 요청자 | Text Input | 발행요청자명 |
| 회사 | Select | 고객사 선택 |
| 종류 | Select | 계산서, 카드, 미발행, 가발행 |
| 상태 | Select | 취소, 요청, 발행 |
| 요청제목 | Text Input | 발행 건 제목 검색 |

## 4. 출력 필드
| 컬럼명 | 설명 |
|--------|------|
| 번호 | 일련번호 |
| 날짜 | 발행요청 연월 |
| 제목 | 발행 건 제목 |
| 고객사 | 청구 대상 회사명 |
| 요청자 | 발행 담당자 |
| 발행액 | 청구 금액 |

## 5. 액션 버튼
- `[조회]` : 필터 적용 후 결과 조회
- `[엑셀 다운로드]` : 조회 결과 Excel 저장
- 제목 클릭 → 발행요청 상세 화면
