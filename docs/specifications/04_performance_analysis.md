# 경영정보 - 경영성과분석 기능명세서

## 1. 개요 및 핵심 기능
선택한 특정 연도/월 기준으로 전사의 내외부 매출, 인건비, 경비, 순수익 등 핵심 회계 지표를 모니터링합니다.

│  [검색 조건]                                                               │

## 3. 입력 필드
| 필드명 | 유형 | 설명 |
|--------|------|------|
| 연도 | Number + 화살표 | `[<]` `[>]` 이동 |
| 월 | Radio Group | 1월 ~ 12월 중 선택 |

## 4. 출력 필드
| 지표명 | 설명 |
|--------|------|
| 매출外 | 외부 고객사 매출액 |
| 매출內 | 사내 내부 매출액 |
| 매입外 | 외부 매입(협력사 등) |
| 매입內 | 사내 내부 매입 |
| 인건비 | 인력 투입에 따른 비용 |
| 경비 | 부대 경비 합계 |
| 순수익 | 최종 영업 이익 |

## 5. 액션 버튼
| 버튼 | 동작 |
|------|------|
| `[<]` / `[>]` | 연도 이동 |
| `[검색]` | 선택 월 기준 데이터 조회 |
| `[내부거래 정보 갱신]` | 사내 거래 데이터 동기화 실행 |
| `[🔄]` | 화면 새로고침 |
