# 서비스업무(Service Operations) 기능분석서 및 화면구성도

## 개요
한마음 시스템(hm.hanmam.kr)의 '서비스업무' 하위 메뉴에 대한 각 화면별 입력값(검색조건), 출력값(그리드/테이블 컬럼), 그리고 주요 액션 버튼을 분석한 문서입니다.

---

## 1. 상담접수 (`/Consult?command=edit`)
* **핵심 기능:** 고객의 상담 요청(장애, 영업, 일반 등)을 시스템에 접수합니다.
* **입력값 (작성 폼):**
  * 사용자
  * 연락처
  * 이메일
  * 상담 분류 (일반상담, 영업문의, 장애문의)
  * 계약명
  * 문의내용 (필수)
  * 첨부파일
  * 연락시한 (날짜, 시, 분)
* **출력값:** 입력 폼 형태로 특정 조회성 출력 목록은 없음
* **주요 버튼:**
  * `[파일선택]`
  * 시한 프리셋: `[즉시]`, `[10분 후]`, `[1시간 후]`, `[금일 중]`
  * `[등록]`

## 2. 상담관리 (`/Consult?command=list`)
* **핵심 기능:** 접수된 상담 내역을 조회하고 처리 상태를 관리합니다.
* **입력값 (검색 조건):**
  * 사용자
  * 계약명 (검색 아이콘)
  * 문의내용
  * 처리상태 (전체, 접수, 완료)
  * 접수일 (시작일 ~ 종료일)
* **출력값 (목록 컬럼):**
  * 번호
  * 사용자
  * 문의내용
  * 계약명
  * 접수시간
  * 접수자
* **주요 버튼:**
  * `[검색(돋보기)]`
  * `[초기화]`
  * 상세조회(개별 항목 클릭)

## 3. 상담통계 (`/Consult?command=resultList`)
* **핵심 기능:** 계약별 또는 월별 상담 실적과 성공률 등 통계 지표를 확인합니다.
* **입력값 (검색 조건):**
  * 연도 선택
  * 월 선택 (1~12월)
  * 계약명
* **출력값 (목록 컬럼):**
  * 순위
  * 고객명
  * 계약금액
  * 상담건수
  * 성공/실패 건수
* **주요 버튼:**
  * 연도 이동 (`[<]`, `[>]`)

## 4. IP관리 (`/Resource?command=list`)
* **핵심 기능:** 사내 IP 대역별 할당 현황과 사용 목적을 관리합니다.
* **입력값 (검색 조건):**
  * IP 대역 선택 (Select Box)
* **출력값 (목록 컬럼):**
  * IP
  * 사용자
  * 목적
  * 발급일
  * 발급자
* **주요 버튼:**
  * 정렬 기능 (컬럼 헤더 클릭)
