# 알림게시판(Notification Board) 기능분석서 및 화면구성도

## 개요
한마음 시스템(hm.hanmam.kr)의 '알림게시판' 하위 메뉴에 대한 각 화면별 입력값(검색조건), 출력값(그리드/테이블 컬럼), 그리고 주요 액션 버튼을 분석한 문서입니다. 전사적인 소통과 자원 예약을 담당합니다.

---

## 1. 관심목록 (`/CallBoard?command=list`)
* **기능:** 사용자가 관심 항목으로 등록한 게시물이나 업무 연락을 모아서 관리합니다.
* **입력값 (검색 조건):**
  * 제목
  * 등록자
* **출력값 (목록 컬럼):**
  * 번호
  * 제목
  * 등록수
  * 작성자
  * 등록일시
* **주요 버튼:**
  * `[검색]`
  * `[선택항목 삭제]`
  * `[등록]`

## 2. 일정공유 (`/Schedule?command=calendar`)
* **기능:** 캘린더 형태의 인터페이스를 통해 개인 및 팀의 일정을 등록하고 공유합니다.
* **입력값 (검색 조건/네비게이션):**
  * 연/월 이동 버튼
  * 오늘(Today) 버튼
* **출력값 (월간 달력):**
  * 일별 등록된 일정 목록
* **주요 버튼:**
  * 이전/다음 달 이동
  * 등록된 일정 클릭 시 상세조회

## 3. 공통 게시판 구역 (공지사항, 자유게시판, 회사게시판, 통합게시판, 한마음 개선요청)
* **기능:** 전사 공지 및 사용자 간 정보 공유를 위한 게시판입니다.
* **입력값 (검색 조건):**
  * 제목
  * 등록일 (시작 ~ 종료)
  * 등록자
* **출력값 (목록 컬럼):**
  * 번호
  * 제목
  * 등록일
  * 작성자
  * 부서명
  * 조회수
* **주요 버튼:**
  * `[검색]`
  * `[선택항목 보기]`
  * `[등록]`

## 4. 자원 예약 (차량예약, 회의실예약)
* **기능:** 회사 공용 자원(차량, 회의실)의 예약 현황을 확인하고 신규 예약을 진행합니다.
* **입력값 (네비게이션):**
  * 연/월 이동 버튼
  * 오늘(Today) 버튼
* **출력값 (월간 달력):**
  * 예약 상태(사용자명, 예약 시간 등)
* **주요 버튼:**
  * 이전/다음 달 이동
  * 특정 날짜 클릭 시 예약 등록 팝업
