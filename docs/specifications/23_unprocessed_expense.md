# 자금업무 - 미처리 지출 관리 기능명세서

## 1. 개요
출금은 완료되었으나, 정산 시스템의 지출 내역과 아직 연결(매칭)되지 않은 미처리 항목을 관리합니다.

## 2. 화면 구성도 (Wireframe)
```
┌──────────────────────────────────────────────────────────────────────┐
│  미처리 지출 관리                                                        │
├──────────────────────────────────────────────────────────────────────┤
│ [검색]                                                                 │
│  2024년  [1]...[12]월  회사: [▼]  담당자: [____]                      │
│                                    [조회] [선택목록 조회] [가상지출 등록]│
├──────────────────────────────────────────────────────────────────────┤
│     ⚠ 매칭되지 않은 출금 내역                                          │
│ 번호│날짜    │지출명         │계정과목   │금액     │출금액   │구분│담당자│
│─────┼────────┼───────────────┼──────────┼─────────┼─────────┼────┼─────│
│  1  │2024-11 │ ?미확인 출금  │           │         │ 200,000 │ ⚠ │     │
└──────────────────────────────────────────────────────────────────────┘
```

## 3. 입력 필드
| 필드명 | 유형 | 설명 |
|--------|------|------|
| 연도 | Number | |
| 월 | Radio Group | |
| 회사 | Select | |
| 담당자 | Text Input | |

## 4. 출력 필드
| 컬럼명 | 설명 |
|--------|------|
| 번호 | |
| 날짜 | 출금 발생일 |
| 지출명 | 매칭된 지출명 (없으면 미매칭 표시) |
| 계정과목 | |
| 금액 | 지출 시스템 금액 |
| 출금액 | 실제 통장 출금액 |
| 구분 | 매칭상태 (완료/⚠미처리) |
| 담당자 | |
| 근거 | 증빙 링크 |

## 5. 액션 버튼
- `[조회]`
- `[선택목록 조회]`
- `[가상지출 등록]` : 시스템 외 지출 항목 수동 등록
