# 자금업무 - 통장 입출금내역 기능명세서

## 1. 개요
등록된 법인 통장의 실제 거래 내역(입금/출금)을 기간별로 모니터링하고, 수금/지급 연결 처리를 합니다.

## 2. 화면 구성도 (Wireframe)
```
┌──────────────────────────────────────────────────────────────────────┐
│  통장 입출금내역                                                         │
├──────────────────────────────────────────────────────────────────────┤
│ [검색]                                                                 │
│  회사: [▼]  통장: [우리은행 (주계좌)▼]                                │
│  기간: [2024-11-01] ~ [2024-11-30]               [조회]               │
├──────────────────────────────────────────────────────────────────────┤
│ [입금보고 작성] [출금보고 작성] [선택목록 조회] [통장거래내역 파일등록]  │
│ 거래일시      │ 입금액      │ 출금액  │ 잔액      │ 차액  │ 내용      │
│───────────────┼─────────────┼─────────┼───────────┼───────┼──────────│
│ 2024-11-05 10:│ 1,100,000   │         │50,100,000 │       │ABC(주)수금│
│ 2024-11-06 14:│             │500,000  │49,600,000 │       │임차료 지급│
└──────────────────────────────────────────────────────────────────────┘
```

## 3. 입력 필드
| 필드명 | 유형 | 설명 |
|--------|------|------|
| 회사 | Select | 법인 선택 |
| 통장 | Select | 등록된 계좌 선택 |
| 기간 | Date Range | 조회 시작일 ~ 종료일 |

## 4. 출력 필드
| 컬럼명 | 설명 |
|--------|------|
| 거래일시 | 거래 발생 일시 |
| 입금액 | 수신 금액 |
| 출금액 | 지급 금액 |
| 잔액 | 거래 후 잔고 |
| 차액 | 예상 대비 차이 |
| 내용 | 적요/거래 설명 |

## 5. 액션 버튼
- `[조회]`
- `[입금보고 작성]` : 수금 보고서 작성 이동
- `[출금보고 작성]` : 지급 보고서 작성 이동
- `[선택목록 조회]` : 선택한 항목만 필터링
- `[통장거래내역 파일등록]` : 은행 내역 파일 업로드
