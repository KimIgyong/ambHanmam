# 서비스업무 - 상담통계 기능명세서

## 1. 개요
계약별 또는 월별 상담 실적과 성공률 등 통계 지표를 확인합니다.

## 2. 화면 구성도 (Wireframe)
```
┌──────────────────────────────────────────────────────────────────────┐
│  상담통계                                                              │
├──────────────────────────────────────────────────────────────────────┤
│  [<] 2024년 [>]   월: [11▼]   계약명: [____🔍]                       │
├──────────────────────────────────────────────────────────────────────┤
│  순위 │ 고객명          │ 계약금액      │ 상담건수 │ 성공  │ 실패    │
│───────┼─────────────────┼───────────────┼──────────┼───────┼─────── │
│  🥇 1 │ (주)ABC         │ 120,000,000   │   45     │  40   │   5    │
│  🥈 2 │ (주)XYZ         │  80,000,000   │   30     │  25   │   5    │
└──────────────────────────────────────────────────────────────────────┘
```

## 3. 입력 필드
| 필드명 | 유형 | 설명 |
|--------|------|------|
| 연도 | Number + 이동버튼 | [<] [>] |
| 월 | Select | 1~12월 |
| 계약명 | Search Input | |

## 4. 출력 필드
| 컬럼명 | 설명 |
|--------|------|
| 순위 | 랭킹 (1~N) |
| 고객명 | |
| 계약금액 | |
| 상담건수 | 총 처리 건수 |
| 성공 | 완료 건수 |
| 실패 | 미해결/반려 건수 |

## 5. 액션 버튼
- 연도 이동 `[<]` `[>]`
