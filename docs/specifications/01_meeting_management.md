# 경영정보 - 회의관리 기능명세서

## 1. 개요 및 핵심 기능
주간/월간 보고 등 부서, 전사 단위의 주요 회의 이력을 전자적으로 조회하고 등록/관리합니다.

## 3. 입력 필드 (Search & Input Form)
| 필드명 | 유형 | 필수 | 설명 |
|--------|------|------|------|
| 제목 | Text Input | | 회의 제목 키워드 검색 |
| 등록자 | Text Input | | 작성자명 검색 |
| 보고일 (시작) | Date Picker | | 보고일 범위 시작 |
| 보고일 (종료) | Date Picker | | 보고일 범위 종료 |

## 4. 출력 필드 (Grid Output)
| 컬럼명 | 설명 |
|--------|------|
| 선택 | 체크박스 (다중 선택) |
| 제목 | 회의 제목 (클릭 시 상세 이동) |
| 보고일 | 보고/회의 실시 일자 |
| 등록일 | 시스템 등록 일자 |
| 작성자 | 보고서 작성자명 |

## 5. 액션 버튼
| 버튼 | 동작 |
|------|------|
| `[검색]` | 조건에 맞는 회의 이력 조회 |
| `[초기화]` | 검색 조건 필드 초기화 |
| `[삭제]` | 체크된 항목 일괄 삭제 |
| `[등록]` | 신규 회의 정보 등록 화면 이동 |
