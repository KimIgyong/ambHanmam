# 영업업무 - 지출예산관리 기능명세서

## 1. 개요
프로젝트별로 승인된 지출 예산 항목을 월별로 관리하고 증빙 내역을 기록합니다.

## 2. 화면 구성도 (Wireframe)
```
┌──────────────────────────────────────────────────────────────────────┐
│  지출예산관리                                                           │
├──────────────────────────────────────────────────────────────────────┤
│ [검색]                                                                 │
│  연도: [2024▼]  월: [11▼]  프로젝트: [🔍____]  담당자: [____]        │
│                                                          [조회]        │
├──────────────────────────────────────────────────────────────────────┤
│  ☐ 번호 │ 날짜    │ 매입 내역       │ 프로젝트명   │ 계정과목          │
│ ────────┼─────────┼─────────────────┼──────────────┼──────────────── │
│  ☐  1  │2024-11  │ AWS 서버 비용   │ A프로젝트    │ 소모품비           │
│  ☐  2  │2024-11  │ 외주 개발 용역  │ B프로젝트    │ 지급수수료         │
└──────────────────────────────────────────────────────────────────────┘
```

## 3. 입력 필드
| 필드명 | 유형 | 설명 |
|--------|------|------|
| 연도 | Select | |
| 월 | Select | |
| 프로젝트 | Search Input | |
| 담당자 | Text Input | |

## 4. 출력 필드
| 컬럼명 | 설명 |
|--------|------|
| 선택 | 체크박스 |
| 번호 | 일련번호 (클릭 시 수정) |
| 날짜 | 발생 연월 |
| 매입 내역 | 지출 항목명 (클릭 시 상세) |
| 프로젝트명 | 귀속 프로젝트 |
| 계정과목 | 회계 분류 |

## 5. 액션 버튼
- `[조회]`
- 매입 내역 클릭 → 상세
- 번호 클릭 → 수정
