### 2.2 REQ-01: 이슈 리스트 상세보기 미구현

**현재 동작:**
- `ProjectIssuesTab.tsx`에서 이슈 목록을 테이블 형태로 표시
- 테이블 행(`<tr>`)에 `onClick` 핸들러 없음
- `IssueDetailModal` 미연동

**코드 확인:**
```tsx
// ProjectIssuesTab.tsx - 현재 상태
{issues.map((issue: IssueResponse) => (
  <tr key={issue.issueId} className="hover:bg-gray-50">   // ← 클릭 이벤트 없음
    ...
  </tr>
))}
```

**비교 기준 (정상 동작 화면):**  
독립 이슈 메뉴(`/issues`)에서는 `IssueListView` → `IssueItem` → 클릭 시 `IssueDetailModal` 열림

### 2.3 REQ-02: 칸반보드 드래그앤드롭 미구현

**현재 동작:**
- `ProjectKanbanTab.tsx`는 정적 카드 표시만 구현
- `@dnd-kit/core` 미사용 → 드래그앤드롭 불가
- 상태 변경 API 연동 없음
- 칸반 컬럼이 4개(`OPEN`, `IN_PROGRESS`, `RESOLVED`, `CLOSED`)만 표시 (APPROVED, REJECTED 누락)
- 카드 클릭 시 상세보기도 없음

**비교 기준 (정상 동작):**  
독립 이슈 메뉴(`/issues`) → 칸반 탭: `IssueKanbanView` 컴포넌트가 이미 완전한 DnD 기능 포함
- `@dnd-kit/core` 기반 드래그앤드롭
- `useUpdateIssueStatus` 훅으로 상태 변경 API 호출
- `ISSUE_VALID_TRANSITIONS` 기반 유효 전환 검증
- 권한 기반 제어 (MANAGER_ONLY_STATUSES)

---

### 2.4 REQ-03: 간트차트 이슈 제목 안 보임

**현재 동작:**
- `ProjectGanttTab.tsx`는 좌측 고정 레이블 없이 바(bar)만 표시하는 방식
- 이슈 제목은 hover 시에만 표시되는 overlay 방식:
  ```tsx
  <div className="absolute left-0 z-10 bg-white/90 ... opacity-0 group-hover:opacity-100 ...">
    <p className="text-xs font-medium text-gray-900 truncate">{issue.title}</p>
  </div>
  ```
- 기본 상태에서는 `opacity-0` → **이슈 제목이 보이지 않음**
- 간트 바(`색상 배경 + text-white`)의 truncate 텍스트가 유일한 표시 방법인데, 바 위치에 따라 왼쪽 여백 영역과 겹치거나 잘림

**비교 기준 (정상 동작):**  
독립 이슈 메뉴의 `IssueGanttView`는 좌측 240px 고정 레이블 영역에 이슈 제목을 항상 표시

### REQ-04: 프로젝트 첫 화면 이슈 리스트로 변경

| 항목 | 내용 |
|------|------|

- 프로젝트 상세 URL 접근 시 기본 활성 탭이 issues

| 항목 | 내용 |

- 상세 모달에서 수정 버튼 클릭 시 이슈 편집 폼 열림
- 상세 모달에서 닫기 후 이슈 목록 갱신

- 드래그앤드롭으로 상태 변경 가능 (유효 전환만 허용)
- 카드 클릭 시 상세 모달 표시

## 5. 사용자 플로우

### REQ-01 사용자 플로우

```
프로젝트 상세 → [이슈 탭] 클릭 → 이슈 테이블 표시
  → 이슈 행 클릭 → IssueDetailModal 열림
    → [수정] 클릭 → IssueFormModal 열림 → 저장 → 목록 갱신
    → [삭제] 클릭 → 삭제 확인 → 목록 갱신
    → [닫기] 클릭 → 모달 닫힘
```

### REQ-02 사용자 플로우

```
프로젝트 상세 → [칸반 탭] 클릭 → 칸반 컬럼 표시 (DnD)
  → 카드 드래그 시작 → 유효 드롭 영역 하이라이트
  → 다른 컬럼에 드롭 → 상태 변경 API 호출 → 카드 이동
  → 카드 클릭 → IssueDetailModal 열림
```

### REQ-03 사용자 플로우

```
프로젝트 상세 → [간트 탭] 클릭 → 좌측 이슈 제목 목록 + 우측 간트 바 표시
  → 좌측 이슈 제목 클릭 or 간트 바 클릭 → IssueDetailModal 열림
```
