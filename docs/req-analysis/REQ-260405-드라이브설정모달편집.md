# REQ-드라이브설정모달편집-20260405

> `/admin/total-users` > Drive Settings 탭에서 Entity Name/Code 클릭 시 모달 편집 화면 제공

## 1. 요구사항 요약

| 항목 | 내용 |
|------|------|
| **요청 위치** | `/admin/total-users` 페이지 → Drive Settings 탭 |
| **핵심 요구** | Entity Name / Entity Code 클릭 → 모달 팝업으로 드라이브 설정 편집 |
| **편집 범위** | 모든 드라이브 설정 항목 수정 가능 |
| **접근 권한** | ADMIN_LEVEL 전용 (기존과 동일) |

### 2.2 현재 테이블 렌더링

```
┌─────────────┬──────────┬──────────┬────────┬──────────────────┬─────────────┬─────────┐
│ Entity Name │ Code     │ 설정됨   │ Source │ 위임 이메일      │ 루트 폴더   │ Actions │
├─────────────┼──────────┼──────────┼────────┼──────────────────┼─────────────┼─────────┤
│ Amoeba HQ   │ AHQ      │ ✓        │ 자체   │ (텍스트)         │ (텍스트)    │ ✏️      │
│ Vietnam 01  │ VN01     │ ✓        │ 자체   │ (인라인 input)   │ (인라인 x2) │ 💾 ✕    │
└─────────────┴──────────┴──────────┴────────┴──────────────────┴─────────────┴─────────┘
```

- **편집 트리거**: Actions 열의 ✏️ (Pencil 아이콘) 클릭
- **편집 방식**: 인라인 — 해당 행의 Impersonate Email, Root Folder ID, Root Folder Name 셀이 `<input>`으로 전환
- **저장**: 행 내 Save 아이콘 → `PUT /api/v1/admin/entities/{entityId}/drive-settings`
- **Entity Name/Code 셀**: 클릭 불가, 단순 텍스트

### 2.4 백엔드 API

| Method | Endpoint | DTO | 비고 |
|--------|----------|-----|------|
| GET | `/api/v1/admin/entities-drive-settings` | — | 전체 엔티티 드라이브 설정 조회 |
| PUT | `/api/v1/admin/entities/{entityId}/drive-settings` | `UpdateDriveSettingsRequest` | upsert 동작 (없으면 생성) |

### 2.5 DB 스키마 (`amb_drive_settings`)

| 컬럼 | 타입 | 설명 |
|------|------|------|
| `drs_id` | UUID PK | 설정 ID |
| `ent_id` | UUID FK nullable | null=전역, UUID=엔티티별 |
| `drs_impersonate_email` | varchar(200) | Google Drive 위임 이메일 |
| `drs_billing_root_folder_id` | varchar(100) | 공유 드라이브/폴더 ID |
| `drs_billing_root_folder_name` | varchar(200) | 공유 드라이브/폴더 이름 |
| `drs_updated_at` | timestamp | 수정일시 |
| `drs_updated_by` | UUID FK | 수정자 |

#### 모달 구성 요소

| 영역 | 내용 |
|------|------|
| 헤더 | HardDrive 아이콘 + Entity Name (Entity Code) + 모달 설명 + 닫기(X) |
| 상태 배지 | Configured 여부 아이콘 + Source 배지(색상) + 최종 수정 일시 |
| 편집 필드 | Impersonate Email (email, max 200) |
| | Billing Root Folder ID (text, mono, max 100) |
| | Billing Root Folder Name (text, max 200) |
| 액션 | 취소 (border btn) + 저장 (violet btn, 로딩 상태) |

### 3.3 기존 인라인 편집 제거
