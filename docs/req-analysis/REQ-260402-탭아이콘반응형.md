# 요구사항 분석서 — 탭/필터 아이콘 반응형 적용

> **문서 번호**: REQ-탭아이콘반응형-20260402  
> **작성일**: 2026-04-02  
> **참조 문서**: `reference/ama-menu-icon-mapping.md`  
> **대상 페이지**: Today, Task(Todo), Issue, Note(MeetingNotes)

---

## 1. 요구사항 요약

`reference/ama-menu-icon-mapping.md`에 정의된 아이콘 매핑을 기준으로,  
Today / Task / Issue / Note 페이지의 탭 및 필터 버튼에 아이콘을 추가한다.  
**가로 사이즈가 줄어들 때(< `sm` breakpoint, 640px)는 텍스트를 숨기고 아이콘만 표시**한다.

---

## 2. AS-IS 현황 분석

### 2-1. 공통 컴포넌트 — ViewScopeTab

**파일**: `apps/web/src/shared/components/ViewScopeTab.tsx`  
**사용처**: TodayPage, TodoPage(Task), IssueFilters(Issue), MeetingNotesPage(Note)

현재 렌더링:
```tsx
// 텍스트만 표시
<button>
  {t(`viewScope.${scope}`)}
</button>
```

현재 탭 텍스트 레이블 (ko):
| scope | 레이블 |
|-------|--------|
| mine | 내것만보기 |
| unit | Unit/부서 보기 |
| cell | Cell/셀 보기 |
| all | 회사 전체 보기 |

**문제**: 좁은 화면에서 텍스트가 그대로 노출되어 레이아웃이 깨지거나 wrapping 발생.  
**문제**: 아이콘 없이 텍스트만 있어 `ama-menu-icon-mapping.md` 매핑 기준 미적용.

---

### 2-2. 이슈 필터 스코프 탭 — IssueFilters SCOPES

**파일**: `apps/web/src/domain/issues/components/IssueFilters.tsx`  
**위치**: 필터 영역 상단 좌측 4개 버튼

현재 SCOPES 배열 및 렌더링:
```tsx
const SCOPES = [
  { key: 'my_involved', labelKey: 'issues:filter.myInvolved' },
  { key: 'my_assigned', labelKey: 'issues:filter.myAssigned' },
  { key: 'my',          labelKey: 'issues:filter.myIssues'   },
  { key: undefined,     labelKey: 'issues:filter.allIssues'  },
] as const;
// 렌더링: 텍스트만 표시
<button>{t(scope.labelKey)}</button>
```

**문제**: 아이콘 없이 텍스트만 있어 좁은 화면에서 레이아웃 깨짐 가능.

---

### 2-3. 노트 타입 필터 탭 — MeetingNotesPage NOTE_TYPE_TABS

**파일**: `apps/web/src/domain/meeting-notes/pages/MeetingNotesPage.tsx`

현재 NOTE_TYPE_TABS 구성:
```tsx
const NOTE_TYPE_TABS = [
  { value: '',           icon: <FileText className="h-3.5 w-3.5" /> },  // All
  { value: 'MEETING_NOTE', icon: <FileText className="h-3.5 w-3.5" /> }, // ← 매핑에서는 Video
  { value: 'MEMO',       icon: <StickyNote className="h-3.5 w-3.5" /> }, // ← 매핑에서는 FileText
  { value: 'DAILY_NOTE', icon: <Calendar className="h-3.5 w-3.5" /> },   // ← 매핑에서는 CalendarDays
];
```

**문제**: `ama-menu-icon-mapping.md`에서 정의한 공식 아이콘과 불일치.  
- `MEETING_NOTE` → `Video` (현재 `FileText`)
- `MEMO` → `FileText` (현재 `StickyNote`)
- `DAILY_NOTE` → `CalendarDays` (현재 `Calendar`)

좁은 화면에서 `flex-wrap`으로 줄바꿈되나 텍스트 숨김 처리는 미적용.

### 2-4. Today 페이지 액션 버튼

**파일**: `apps/web/src/domain/today/pages/TodayPage.tsx`

현재 우측 액션 버튼 아이콘:
| 버튼 | 현재 아이콘 | 매핑 기준 아이콘 |
|------|------------|----------------|
| AI 업무 코칭 | `Sparkles` | `BrainCircuit` |
| 저장된 리포트 | `FileText` | `BookMarked` |
| 업무 리포트 | `FileBarChart` | `BarChart2` |

이미 `<span className="hidden sm:inline">` 패턴으로 반응형 처리되어 있음.  
**문제**: 아이콘이 `ama-menu-icon-mapping.md` 공식 매핑과 다름.

### 2-5. MyCellIcon 커스텀 컴포넌트 미존재

`ama-menu-icon-mapping.md`에서는 `Hexagon` + 내부 `Circle`을 합성한 커스텀 `MyCellIcon` 사용을 지정하였으나,  
`apps/web/src/components/common/icons/MyCellIcon.tsx` 파일이 존재하지 않음.

## 3. TO-BE 요구사항

### 3-1. ViewScopeTab 반응형 아이콘 적용

- 각 scope 버튼에 아이콘 + 텍스트 동시 렌더링
- `sm` 미만(< 640px): 텍스트 숨기고 아이콘만 표시
- `sm` 이상(≥ 640px): 아이콘 + 텍스트 동시 표시

아이콘 매핑:
| scope | 아이콘 컴포넌트 | 비고 |
|-------|---------------|------|
| mine | `UserCircle2` | My Only |
| unit | `Users` | My Unit |
| cell | `MyCellIcon` | 커스텀 (Hexagon + 내부 Circle) |
| all | `LayoutGrid` | All |

---

### 3-2. IssueFilters SCOPES 반응형 아이콘 적용

- 각 scope 버튼에 아이콘 + 텍스트 동시 렌더링
- `sm` 미만: 텍스트 숨기고 아이콘만 표시

아이콘 매핑:
| scope key | 아이콘 컴포넌트 | 레이블 |
|-----------|---------------|--------|
| my_involved | `Users2` | My Involved |
| my_assigned | `UserCheck` | Assigned Me |
| my | `Target` | My Issues |
| undefined (All) | `LayoutGrid` | All |

---

### 3-3. MeetingNotesPage NOTE_TYPE_TABS 공식 아이콘 적용

공식 아이콘으로 교체 및 반응형 텍스트 처리:
| value | 아이콘 컴포넌트 | 비고 |
|-------|---------------|------|
| '' (All) | `FileText` | 변경 없음 |
| MEETING_NOTE | `Video` | `FileText` → `Video` 변경 |
| MEMO | `FileText` | `StickyNote` → `FileText` 변경 |
| DAILY_NOTE | `CalendarDays` | `Calendar` → `CalendarDays` 변경 |

---

### 3-4. TodayPage 액션 버튼 공식 아이콘 적용

| 버튼 | 교체 아이콘 |
|------|-----------|
| AI 업무 코칭 | `Sparkles` → `BrainCircuit` |
| 저장된 리포트 | `FileText` → `BookMarked` |
| 업무 리포트 | `FileBarChart` → `BarChart2` |

---

### 3-5. MyCellIcon 커스텀 컴포넌트 신규 생성

`apps/web/src/components/common/icons/MyCellIcon.tsx` 파일 생성.  
`ama-menu-icon-mapping.md` 명세대로 Hexagon polygon + 내부 circle SVG 합성.

## 5. 사용자 플로우

```
[좁은 화면 < 640px]
  탭 버튼 → 아이콘만 표시, tooltip으로 레이블 확인 가능

[일반 화면 ≥ 640px]
  탭 버튼 → 아이콘 + 텍스트 함께 표시 (현재와 유사)
```

## 6. 기술 제약사항

- `lucide-react` 현재 설치 버전: `^0.356.0` (BrainCircuit 포함, v0.338+ 충족)
- 반응형 처리: TailwindCSS `sm:` breakpoint (640px) 기준
- `hidden sm:inline` 패턴은 TodayPage 기존 버튼에서 이미 사용 중 — 동일 패턴 적용
- MyCellIcon: `SVGProps<SVGSVGElement>` 기반 인터페이스 사용
- ViewScopeTab은 공유 컴포넌트 → 변경 시 Today/Task/Issue/Note 4개 페이지 동시 적용됨
