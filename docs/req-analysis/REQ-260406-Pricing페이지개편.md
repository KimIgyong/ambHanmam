## 2. AS-IS 현황 분석

### 현재 PricingPage 구조
1. **Hero Header** — 제목/소제목
2. **Plan Cards (3장)** — Free / Basic / Premium 카드형 비교
3. **Comparison Table** — 11행 비교표 (단순 텍스트)
4. **FAQ** — 5개 항목
5. **Bottom CTA** — 가입 유도

### 현재 데이터 문제점
- 비교표가 단순 텍스트만 나열, 핵심 차별 포인트 부족
- BASIC 구간별 요금 상세 없음
- 연간 결제 할인 비교 섹션 없음
- 레퍼럴 보상 섹션 없음
- 스토리지 정책 섹션 없음
- ADD-ON 단가 섹션 없음
- 토큰 리셋 정책(Free: 이월 / Basic: 초기화) 미표기

## 3. TO-BE 요구사항 (이미지 기준)

### Section 1: 메인 요금제 비교 (ama-price-01)
| 비교 항목 | FREE | BASIC | PREMIUM |
|----------|------|-------|---------|

| 사용자 구성 | 최대 5명 (오너1+직원4) | 6~49명 (5인 단위 증가, 유료인원×$2) | 50명 이상 |
| 월 구독료 | $0 | 유료인원×$2 (최소 $10 ~ $88) | 상담 후 결정 |
| AI 토큰 기본 | 50,000T (가입 시 1회 지급) | 1인당 20,000T / 월 (전체 공유 풀) | 상담 후 결정 |
| 토큰 추가 방식 | 1만T 단위 구매 ($1/10K) | 5만T 단위 충전 ($5/1회, 소진 즉시 가능) | 상담 후 결정 |
| 토큰 월 리셋 | **없음** (구매 토큰도 이월) | **매월 초기화** (이월 없음, 월 1일 리셋) | 상담 후 결정 |

| 스토리지 추가 | 불가 | 5GB 단위 · $1/GB | 상담 후 결정 |
| 스토리지 상한 | 1GB 고정 | 최대 100GB (기본 3G 포함 총 상한) | 상담 후 결정 |
| 레퍼럴 보상 | 해당 없음 | 50,000T 지급 (10명 초대 메일 발송 시) | 상담 후 결정 |
| 연간 할인 | - | 2개월 무료 (연간 선결제 시) | ✓ |

> **Premium 별도 견적 케이스**: ① 50명 이상 ② 온프레미스 ③ 자체 AI 모델 연동 ④ 전용 SLA/보안 감사
