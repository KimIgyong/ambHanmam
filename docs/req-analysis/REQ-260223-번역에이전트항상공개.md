## 1. AS-IS 현황

### 1-1. 에이전트 노출 방식 (기존)
- AI 어시스턴트 모달의 에이전트 목록은 `GET /api/v1/agent-configs/visible` API 응답 기반
- `getVisibleAgents()` 서비스 내부 로직:
  - `agcIsActive = true` 인 에이전트만 조회
  - `agcVisibleGroupIds` 설정에 따라 필터링
    - `null` → 전체 공개
    - `[]` → 비공개
    - `[grpId, ...]` → 해당 그룹만 공개
- **번역(TRANSLATION) 에이전트도 위 필터 로직에 포함** → 관리자가 `is_active=false` 또는 그룹 제한 설정 시 일부 사용자에게 미노출될 수 있었음

### 1-2. 현황 분석 시 발견된 문제
- 에이전트 설정 페이지(`/settings/agents`)에서 TRANSLATION 에이전트의 Active 체크박스 비활성화 처리 없음
- TRANSLATION 에이전트에도 그룹 노출 설정 UI가 표시됨
- 관리자 실수로 번역 에이전트를 비공개로 설정 가능한 상태
