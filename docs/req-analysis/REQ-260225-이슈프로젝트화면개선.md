### 2.2 이슈 수정 화면 - Members API 401 오류

| 항목 | 현황 |
|------|------|
| **에러** | `GET https://mng.amoeba.site/api/v1/members` 401 (Unauthorized) |
| **호출 경로** | `IssueFormModal` → `AssigneeSelector` → `useMemberList()` → `membersService.getMembers()` |
| **인증** | `EntityGuard` + `@Roles('USER')` 적용 |
| **조건** | `enabled: !!entityId` — Entity 선택 시에만 호출 |

**분석 결과**: 
- `GET /members`는 `@Roles('USER')` — 일반 인증 사용자 접근 가능
- 401은 **토큰 만료** 또는 **X-Entity-Id 헤더 누락** 가능성이 높음
- `apiClient`의 인터셉터에서 `X-Entity-Id` 헤더가 정상 전달되는지 확인 필요
- 일시적 인증 문제일 가능성 (재현 확인 필요)
