## 2. AS-IS 현황 분석

### 2.1 R1 — 모바일 PWA 노치 safe-area

**현재 상태:**
- `globals.css`에 CSS 변수 `--sat`, `--sab` 등 safe-area-inset 정의 완료
- 채팅 페이지 내부 모달/토스트에는 별도 safe-area 패딩 미적용
- 모달: `CreateChannelModal`, `InviteMemberModal`, `NewDmModal` — `fixed inset-0 z-50` 사용 중이나 `pt-[var(--sat)]` 미적용
- 토스트: `sonner` 라이브러리 사용 — App.tsx Toaster 컴포넌트에 safe-area 패딩 적용 여부 확인 필요

### 2.2 R2 — 모바일 입력 포커싱 시 가로 사이즈

**현재 상태:**
- `meta viewport` 태그에 `width=device-width, initial-scale=1` 설정
- iOS에서 textarea 포커스 시 자동 줌이 발생할 수 있음 (font-size < 16px일 때)
- 현재 textarea CSS: `text-sm` (14px) → iOS auto-zoom 유발 가능
- `maximum-scale=1, user-scalable=no` 미설정

### 2.3 R3 — 파일/이미지 버튼 배치

### 2.4 R4 — 동시번역 메시지 줄바꿈 삭제

**현재 형식 (백엔드 message.service.ts):**
```
{원문}\n\n---\n🌐 [{언어명}] {번역문}
```
- `\n\n` (빈 줄) + `---` (수평선) + `\n🌐` 형태로 구분

**목표 형식:**
```
{원문}\n---\n[{국가코드}] {번역문}
```
- `\n\n` → `\n` (빈 줄 제거)
- `🌐` 아이콘 삭제
- 언어명 → 2자리 국가코드

### 2.5 R5 — 번역 언어 표시 변경

**현재 상태:**
- 백엔드 `LANG_LABELS`: `{ en: 'English', ko: '한국어', vi: 'Tiếng Việt', ja: '日本語', zh: '中文' }`
- 동시번역 메시지에 `🌐 [English]` 형태로 삽입
- `TranslationMessage.tsx`: Languages 아이콘(🌐) + translatedByUser 텍스트 표시

**목표:**
- `LANG_LABELS` → `{ en: 'EN', ko: 'KO', vi: 'VI', ja: 'JA', zh: 'ZH' }`
- 🌐 아이콘 삭제
- 메시지 포맷: `[EN] translated text`

- en: "Type a message..."

**목표 값:**

### 2.7 R7 — 채널 이름/설명 수정

**현재 상태:**
- 백엔드 PATCH `/talk/channels/:id` 엔드포인트 **구현 완료** (`UpdateChannelRequest` DTO: `name?`, `description?`)
- 프론트엔드 `useUpdateChannel()` 훅 **구현 완료**
- `talk.service.ts` `updateChannel(id, data)` **구현 완료**
- **UI 없음** — 채널 수정 모달/폼이 존재하지 않음
- `ChannelHeader` — 채널명 표시만 하고 수정 인터렉션 없음

**필요 작업:**
- 채널 헤더에서 채널명 클릭 시 수정 모달 표시 (채널 소유자만)
- `EditChannelModal` 컴포넌트 신규 생성

### 2.8 R8 — 동료/crew 온라인 상태

**현재 상태: ✅ 이미 구현 완료**
- 백엔드: `PresenceController` + `PresenceService` (하트비트 30초, 60초 idle → offline)
- 프론트엔드: `usePresenceHeartbeat()`, `usePresenceStatus()`, `usePresenceSSE()`
- UI: `MemberPanel`에 초록/회색 온라인 상태 dot, `ChannelHeader`에 DM 상대방 상태 텍스트
- **채널 목록(ChannelList)에는 미표시** — DM 채널 옆에 온라인 indicator 없음

**필요 작업:**
- `ChannelList`의 DM 채널 항목에 상대방 온라인 상태 dot 표시

## 4. 기술 제약사항

1. **R4 (번역 포맷 변경)**: 기존에 `\n\n---\n🌐` 포맷으로 저장된 메시지는 소급 변경 불가. 프론트엔드 렌더링에서 두 가지 포맷 모두 처리 필요
2. **R2 (iOS auto-zoom)**: `user-scalable=no`는 접근성 가이드라인에 위배되므로, `font-size: 16px` 적용으로 해결하는 것이 권장 방침
3. **R7 (채널 수정)**: 채널 소유자(OWNER)만 수정 가능해야 함 — 백엔드에 이미 소유자 검증 로직 존재
4. **R1 (노치 여백)**: `env(safe-area-inset-top)` 값은 PWA standalone 모드에서만 적용됨
