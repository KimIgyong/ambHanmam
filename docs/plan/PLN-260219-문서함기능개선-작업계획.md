## STEP 1: 백엔드 - 폴더 생성 엔드포인트 + KMS 이벤트 발행

### 대상 파일
- `apps/api/src/domain/drive/controller/drive.controller.ts`
- `apps/api/src/domain/drive/service/drive.service.ts`
- `apps/api/src/domain/drive/drive.module.ts`

### 작업 내용
1. `POST /drive/folders/create` 엔드포인트 (Admin only)
   - Body: `{ parent_folder_id, folder_name }`
   - `GoogleDriveService.createFolder()` 호출
2. `DriveService.uploadFile()` 에서 `ModuleDataEvent` 발행
   - `module: 'drive'`, `type: 'DOC'`
   - content: 파일명 + 폴더 정보
   - `EventEmitter2` 주입 필요

### 사이드 임팩트
- KMS WorkItemSyncListener가 이벤트를 수신하여 WorkItem 생성 및 자동 태깅
- 기존 Drive 읽기 API에 영향 없음
