## 2. 단계별 구현 계획

### Step 1: 백엔드 — updateConfig TRANSLATION 보호 ✅

**파일**: `agent-config.service.ts` → `updateConfig()` 메서드

**변경 전**:
```ts
if (dto.is_active !== undefined) config.agcIsActive = dto.is_active;
if (dto.visible_group_ids !== undefined) config.agcVisibleGroupIds = dto.visible_group_ids;
```

**변경 후**:
```ts
// TRANSLATION 에이전트는 항상 활성화 & 전체 공개 강제
if (deptCode === 'TRANSLATION') {
  config.agcIsActive = true;
  config.agcVisibleGroupIds = null;
} else {
  if (dto.is_active !== undefined) config.agcIsActive = dto.is_active;
  if (dto.visible_group_ids !== undefined) config.agcVisibleGroupIds = dto.visible_group_ids;
}
```

**효과**: API 호출로 TRANSLATION을 비활성화하거나 그룹 제한해도 무시됨
