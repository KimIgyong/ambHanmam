### Phase 5: Admin MenuGuideTab — 법인 선택 추가

**작업 파일:** `apps/web/src/domain/settings/components/MenuGuideTab.tsx`

**현재 문제:** `useEntityStore().currentEntity`가 Admin 컨텍스트에서 null → Tips 저장/조회 불가

**수정 방향:**
1. `useEntities()` 훅으로 전체 법인 목록 로드
2. 드롭다운 state 추가: `const [selectedEntityId, setSelectedEntityId] = useState<string | null>(null)`
3. `useMenuTips(selectedEntityId)`, `useUpsertMenuTip(selectedEntityId)` 에 선택된 entityId 전달

**훅 시그니처 수정 필요:**
```typescript
// useSiteConfig.ts
export function useMenuTips(entityId?: string) {
  const storeEntityId = useEntityStore((s) => s.currentEntity?.entityId);
  const resolvedEntityId = entityId || storeEntityId;
  return useQuery({
    queryKey: ['entity-settings', 'menu-tips', resolvedEntityId],
    queryFn: () => apiClient.get('/entity-settings/site-config/menu-tips', {
      params: resolvedEntityId ? { entity_id: resolvedEntityId } : undefined,
    }).then((r) => r.data.data),
    enabled: !!resolvedEntityId,
  });
}
```

**MenuGuideTab 드롭다운 추가:**
```tsx
import { useEntities } from '@/domain/hr/hooks/useEntity';

const { data: entities = [] } = useEntities();
const [selectedEntityId, setSelectedEntityId] = useState<string>('');

// selectedEntityId가 없으면 첫 번째 entity로 자동 선택
useEffect(() => {
  if (!selectedEntityId && entities.length > 0) {
    setSelectedEntityId(entities[0].entityId);
  }
}, [entities]);

// 드롭다운 UI
<select
  value={selectedEntityId}
  onChange={(e) => setSelectedEntityId(e.target.value)}
  className="rounded-lg border border-gray-300 px-3 py-1.5 text-sm"
>
  {entities.map((e) => (
    <option key={e.entityId} value={e.entityId}>{e.entityName}</option>
  ))}
</select>
```

## 4. 사이드 임팩트 분석

| 변경 | 영향 | 위험도 |
|------|------|--------|
| PageTitle `text-gray-400` 적용 | 이미 PageTitle 사용 중인 모든 페이지 아이콘 색 변경 | 낮음 (의도된 변경) |
| Drive/Expense/Asset/Accounting 헤더 교체 | 기존 배경박스 완전 제거 | 낮음 (시각적 변경만) |
| ProjectLayout PageTitle 추가 | 프로젝트 탭 위에 제목 row 추가 → 레이아웃 높이 증가 | 낮음 |
| HrLayout/BillingLayout PageTitle 추가 | 자식 페이지 자체 제목과 중복 가능 | **중간** → 자식 페이지 제목 중복 확인 필요 |
| useMenuTips 시그니처 변경 | MenuGuideTab 외 다른 곳에서도 사용 중인지 확인 필요 | 낮음 (현재 MenuGuideTab에서만 사용) |
