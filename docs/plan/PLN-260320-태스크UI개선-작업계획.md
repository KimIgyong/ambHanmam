# 작업계획서: Tasks UI 개선 - 완료/이슈등록 버튼 배치 및 확인 다이얼로그

**작성일**: 2026-03-20

## 요구사항 요약

| # | 요구사항 | 대상 |
|---|---------|------|
| 1 | 완료 체크박스 + 이슈등록 아이콘을 **제목 앞**에 배치 | 카드뷰 + 리스트뷰 |
| 2 | 완료 체크박스 클릭 시 confirm: "이 태스크를 완료 처리하겠습니까?" | 전체 |
| 3 | 이슈 등록 아이콘 클릭 시 confirm: "이 태스크를 이슈로 등록하시겠습니까?" | 전체 |
| 4 | 리스트뷰에도 카드뷰와 동일한 액션 버튼 노출 | 리스트뷰 |
| 5 | 히스토리/삭제 버튼은 라인 오른쪽 끝 배치 | 카드뷰 + 리스트뷰 |

## AS-IS / TO-BE

### 카드뷰 (TodoItem.tsx)
- **AS-IS**: 완료 체크박스 없음. 이슈등록/히스토리/삭제가 배지 영역 하단에 위치
- **TO-BE**: 체크박스 + 이슈등록 → 제목 좌측. 히스토리 + 삭제 → 우측 끝

### 리스트뷰 (TodoPage.tsx, ScopedTodoList.tsx)
- **AS-IS**: 상태배지 + 제목 + 태그 + 날짜만 표시, 액션 버튼 없음
- **TO-BE**: 체크박스 + 이슈등록 → 제목 앞. 히스토리 + 삭제 → 우측 끝
