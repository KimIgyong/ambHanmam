## 1. 시스템 개발 현황 분석

### 1.1 백엔드 (변경 불필요)

프로젝트-고객사 배정 API 3개가 이미 `project.controller.ts`에 완전 구현:
- `GET /project/projects/:id/clients` — 배정 고객사 목록 (관계 포함)
- `POST /project/projects/:id/clients` — 배정 추가 (`{ cli_id }`)
- `DELETE /project/projects/:id/clients/:clientId` — 배정 해제

고객 포털 이슈 API도 완전 구현.

## 2. 단계별 구현 계획

### Step 1: project.service.ts — API 메서드 추가

**파일**: `apps/web/src/domain/project/service/project.service.ts`

추가할 메서드:
```typescript
// Clients
getProjectClients = (projectId: string) =>
  apiClient
    .get<ListResponse<any>>(`${this.basePath}/${projectId}/clients`)
    .then((r) => r.data.data);

addProjectClient = (projectId: string, cliId: string) =>
  apiClient
    .post<SingleResponse<any>>(`${this.basePath}/${projectId}/clients`, { cli_id: cliId })
    .then((r) => r.data.data);

removeProjectClient = (projectId: string, clientId: string) =>
  apiClient.delete(`${this.basePath}/${projectId}/clients/${clientId}`);
```

### Step 2: useProject.ts — React Query 훅 추가

**파일**: `apps/web/src/domain/project/hooks/useProject.ts`

추가할 훅:
- `useProjectClients(projectId)` — 배정 고객사 목록 조회
- `useAddProjectClient()` — 고객사 배정 뮤테이션
- `useRemoveProjectClient()` — 배정 해제 뮤테이션
