## 2. 단계별 구현 계획

### Phase 1: 버그 수정 (P0 — 즉시)

#### Task 1-1: 모바일 드로어 카테고리 클릭 버그 수정

**파일**: `apps/web/src/layouts/MainLayout.tsx`

**문제 코드** (현재):
```tsx
// renderMenuItems() 내부
<button
  onClick={() => navigate('/menu/work-tools')}
  className="..."
>
  {t('common:sidebar.workTools')}
</button>
```

**수정 후**:
```tsx
<button
  onClick={() => {
    navigate('/menu/work-tools');
    if (inDrawer) setDrawerOpen(false);  // ← 추가
  }}
  className="..."
>
  {t('common:sidebar.workTools')}
</button>
```
> Work Modules, Custom Apps 카테고리 동일 적용 (총 3곳)

---

### Phase 2: 카드뷰 페이지 탭 내비게이션 추가 (P1)

#### Task 2-1: MenuCategoryPage 탭 UI 추가

**파일**: `apps/web/src/domain/menu/pages/MenuCategoryPage.tsx`

**변경 내용**:
- 페이지 헤더 하단에 3개 탭 추가
- 탭 클릭 시 `navigate('/menu/work-tools')` 등으로 이동
- 현재 페이지 탭이 활성 스타일로 강조

**탭 구성**:
```tsx
const tabs = [
  { type: 'workTools',   path: '/menu/work-tools',   labelKey: 'common:sidebar.workTools' },
  { type: 'workModules', path: '/menu/work-modules',  labelKey: 'common:sidebar.workModules' },
  { type: 'customApps',  path: '/menu/custom-apps',   labelKey: 'common:sidebar.customApps' },
];
```

**UI 스펙**:
```
┌─────────────────────────────────────┐
│ ← [홈]          Work Tools    [10]  │  ← 헤더
├─────────────────────────────────────┤
│ [Work Tools ★] [Work Modules] [Custom Apps] │  ← 탭
├─────────────────────────────────────┤
│  [카드] [카드] [카드] [카드]         │  ← 카드 그리드
│  [카드] [카드] [카드] [카드]         │
└─────────────────────────────────────┘
```

**탭 스타일(TailwindCSS)**:
- 활성 탭: `border-b-2 border-indigo-600 text-indigo-600 font-semibold`
- 비활성 탭: `text-gray-500 hover:text-gray-700`
- 탭 컨테이너: `flex border-b border-gray-200 bg-white px-4`
