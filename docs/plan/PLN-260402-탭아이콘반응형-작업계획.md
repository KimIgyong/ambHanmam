## 2. 단계별 구현 계획

### Phase 1 — MyCellIcon 커스텀 컴포넌트 생성 (신규)

**파일**: `apps/web/src/components/common/icons/MyCellIcon.tsx` (신규 생성)

`ama-menu-icon-mapping.md` 명세 그대로 구현:
- `SVGProps<SVGSVGElement>` 기반 인터페이스
- size/color prop 지원
- SVG: Hexagon polygon + 내부 Circle (cx=12, cy=12, r=2.5)

```tsx
import type { SVGProps } from 'react';

interface MyCellIconProps extends SVGProps<SVGSVGElement> {
  size?: number;
}

const MyCellIcon = ({ size = 18, color = 'currentColor', ...props }: MyCellIconProps) => (
  <svg
    width={size}
    height={size}
    viewBox="0 0 24 24"
    fill="none"
    stroke={color}
    strokeWidth={1.7}
    strokeLinecap="round"
    strokeLinejoin="round"
    {...props}
  >
    <polygon points="12,2.5 20,7 20,17 12,21.5 4,17 4,7" />
    <circle cx="12" cy="12" r="2.5" />
  </svg>
);

export default MyCellIcon;
```

**사이드 임팩트**: 없음 (신규 파일)

### Phase 2 — ViewScopeTab 아이콘 + 반응형 처리

**파일**: `apps/web/src/shared/components/ViewScopeTab.tsx`

**변경 내용**:
1. lucide-react에서 `UserCircle2`, `Users`, `LayoutGrid` import 추가
2. `MyCellIcon` import
3. `SCOPE_ICONS` 맵 정의:
   ```ts
   const SCOPE_ICONS: Record<ViewScope, React.ReactNode> = {
     mine: <UserCircle2 className="h-4 w-4 shrink-0" />,
     unit: <Users className="h-4 w-4 shrink-0" />,
     cell: <MyCellIcon size={16} />,
     all:  <LayoutGrid className="h-4 w-4 shrink-0" />,
   };
   ```
4. 버튼 렌더링 변경:
   ```tsx
   <button ...>
     {SCOPE_ICONS[scope]}
     <span className="hidden sm:inline">{t(`viewScope.${scope}`)}</span>
   </button>
   ```
5. 버튼 클래스에 `gap-1.5` 추가 (아이콘-텍스트 간격)

**사이드 임팩트**: ViewScopeTab을 사용하는 **4개 페이지** (Today, Task, Issue, Note) 자동 반영.  
레이아웃 파편화 없음 — 아이콘 추가로 버튼 너비 약간 증가하나 `flex-wrap` 또는 `flex justify-between` 컨테이너 내부라 문제 없음.

### Phase 3 — IssueFilters SCOPES 아이콘 + 반응형 처리

**파일**: `apps/web/src/domain/issues/components/IssueFilters.tsx`

**변경 내용**:
1. lucide-react에서 `Users2`, `UserCheck`, `Target`, `LayoutGrid` import 추가
2. `SCOPES` 배열에 `icon` 필드 추가:
   ```ts
   const SCOPES = [
     { key: 'my_involved', labelKey: 'issues:filter.myInvolved', icon: <Users2 className="h-3.5 w-3.5 shrink-0" /> },
     { key: 'my_assigned', labelKey: 'issues:filter.myAssigned', icon: <UserCheck className="h-3.5 w-3.5 shrink-0" /> },
     { key: 'my',          labelKey: 'issues:filter.myIssues',   icon: <Target className="h-3.5 w-3.5 shrink-0" /> },
     { key: undefined,     labelKey: 'issues:filter.allIssues',  icon: <LayoutGrid className="h-3.5 w-3.5 shrink-0" /> },
   ] as const;
   ```
3. 버튼 렌더링에 아이콘 + `gap-1` + `hidden sm:inline` 텍스트 처리:
   ```tsx
   <button className={`... flex items-center gap-1`}>
     {scope.icon}
     <span className="hidden sm:inline">{t(scope.labelKey)}</span>
   </button>
   ```

**사이드 임팩트**: Issues 페이지 필터 탭만 영향. 기능 변경 없음.
