# PLN-260413 이슈 코멘트 고객사 공개 제어

## 1. 시스템 현황 분석

### 1.1 백엔드 (apps/api)
- **엔티티**: `IssueCommentEntity` → `amb_issue_comments` 테이블, 공개 여부 컬럼 없음
- **내부 API**: `IssueController` + `IssueService` — 코멘트 CRUD, 공개 제어 없음
- **클라이언트 API**: `ClientIssueController` + `ClientIssueService` — `findIssueById()`에서 모든 코멘트 반환
- **DTO**: `CreateIssueCommentRequest` → `{ content, parent_id? }`, 공개 옵션 없음
- **타입**: `IssueCommentResponse` (packages/types) → `clientVisible` 필드 없음

### 1.2 프론트엔드 (apps/web)
- **내부 UI**: `IssueDetailModal.tsx` — RichTextEditor + 코멘트 목록, 공개 옵션 없음
- **클라이언트 UI**: `ClientIssueDetailPage.tsx` — 모든 코멘트 렌더링, 필터 없음
- **Hook**: `useAddIssueComment` → `{ issueId, content, parentId }`
- **Service**: `issueService.addIssueComment()` → `{ content, parent_id? }`

## 2. 단계별 구현 계획

### Phase 1: DB 엔티티 + 마이그레이션 SQL
- `IssueCommentEntity`에 `isc_client_visible` BOOLEAN 컬럼 추가 (default: false)
- 마이그레이션 SQL 작성 (스테이징/프로덕션 수동 실행용)

### Phase 2: 백엔드 API 수정
- DTO: `CreateIssueCommentRequest`에 `client_visible` 추가
- 타입: `IssueCommentResponse`에 `clientVisible` 추가
- Mapper: `toCommentResponse()`에 `clientVisible` 매핑
- Service (`IssueService`):
  - `addIssueComment()`에 `clientVisible` 파라미터 추가
  - `toggleClientVisible()` 신규 메서드 추가
- Controller (`IssueController`):
  - `addIssueComment()`에서 DTO의 `client_visible` 전달
  - `PATCH :id/comments/:commentId/client-visible` 신규 엔드포인트
- Service (`ClientIssueService`):
  - `findIssueById()`: `client_visible = true` 필터링
  - `addComment()`: 자동 `client_visible = true`

### Phase 3: 프론트엔드 (내부 사용자)
- Service: `addIssueComment()`에 `client_visible` 추가, `toggleClientVisible()` 신규
- Hook: `useAddIssueComment` 확장, `useToggleClientVisible` 신규
- UI (`IssueDetailModal.tsx`):
  - 코멘트 작성 영역에 "고객사에게 공개" 체크박스
  - 코멘트 목록에 공개/비공개 뱃지 + 토글 버튼

### Phase 4: i18n 번역 키 추가
- ko/en/vi issues.json에 코멘트 공개 관련 키 추가

## 5. DB 마이그레이션

```sql
-- 260413: Issue comment client visibility
ALTER TABLE amb_issue_comments 
  ADD COLUMN isc_client_visible BOOLEAN NOT NULL DEFAULT false;

-- 기존 CLIENT_LEVEL 사용자가 작성한 코멘트만 공개 전환
UPDATE amb_issue_comments 
SET isc_client_visible = true 
WHERE isc_author_id IN (
  SELECT usr_id FROM amb_users WHERE usr_level = 'CLIENT_LEVEL'
);
```
