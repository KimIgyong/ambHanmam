## 2. 단계별 구현 계획

### Phase 1: i18n 키 추가

**대상 파일**: `apps/web/src/locales/{ko,en,vi}/totalUsers.json`

추가 키:

| 키 | ko | en | vi |
|----|----|----|-----|
| `drive.editTitle` | 드라이브 설정 편집 | Edit Drive Settings | Chỉnh sửa cài đặt Drive |
| `drive.editDescription` | 엔티티 드라이브 연동 설정을 편집합니다 | Edit entity drive integration settings | Chỉnh sửa cài đặt kết nối Drive của tổ chức |
| `drive.folderId` | 루트 폴더 ID | Root Folder ID | ID thư mục gốc |
| `drive.folderName` | 루트 폴더 이름 | Root Folder Name | Tên thư mục gốc |

### Phase 2: 탭 컴포넌트 수정 (`EntityDriveSettingsTab.tsx`)

**변경 사항**:

1. **인라인 편집 로직 제거**
   - `EditState` 인터페이스 삭제
   - `editingEntity` state 삭제
   - `handleEdit`, `handleSave`, `handleCancel` 함수 삭제
   - 행 내 input 조건부 렌더링 제거

2. **모달 연동 state 추가**
   ```typescript
   const [selectedItem, setSelectedItem] = useState<EntityDriveSettingsItem | null>(null);
   ```

3. **Entity Name/Code 셀 클릭 핸들러**
   ```tsx
   <td className="cursor-pointer ... hover:text-violet-600 hover:underline"
       onClick={() => setSelectedItem(item)}>
     {item.entityName}
   </td>
   ```

4. **Actions 열 간소화**
   - 기존 Save/Cancel 토글 제거
   - Pencil 아이콘 클릭 → `setSelectedItem(item)` (모달 트리거)

5. **모달 렌더**
   ```tsx
   {selectedItem && (
     <DriveSettingsEditModal
       item={selectedItem}
       onClose={() => setSelectedItem(null)}
     />
   )}
   ```

6. **불필요 import 정리**
   - 제거: `Save`, `X` (인라인 편집 전용)
   - 추가: `DriveSettingsEditModal` import
