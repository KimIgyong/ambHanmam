## 3. 단계별 구현 계획

### 단계 1: TodoPage.tsx mine 탭 레이아웃 변경

#### 3.1.1 2칸 그리드 적용
기존 `SECTION_ORDER` 순회 대신, 왼쪽/오른쪽 칸으로 분리하여 렌더링한다.

**변경 전:**
```tsx
const SECTION_ORDER: TodoComputedStatus[] = ['OVERDUE', 'DUE_TODAY', 'IN_PROGRESS', 'UPCOMING_SOON', 'UPCOMING'];

<div className="space-y-4">
  {SECTION_ORDER.map((status) => { ... })}
</div>
```

**변경 후:**
```tsx
const LEFT_SECTIONS: TodoComputedStatus[] = ['OVERDUE', 'DUE_TODAY', 'IN_PROGRESS'];
const RIGHT_SECTIONS: TodoComputedStatus[] = ['UPCOMING_SOON', 'UPCOMING'];

<div className="grid grid-cols-1 gap-4 md:grid-cols-2">
  {/* 왼쪽: 긴급/진행 중 */}
  <div className="space-y-4">
    {LEFT_SECTIONS.map((status) => { ... 기존 렌더링 로직 ... })}
  </div>
  {/* 오른쪽: 향후 예정 */}
  <div className="space-y-4">
    {RIGHT_SECTIONS.map((status) => { ... 기존 렌더링 로직 ... })}
  </div>
</div>
```

#### 3.1.2 Completed 섹션 간소화
- 기존 접힘/펼침 토글 + TodoItem 렌더링 제거
- 숫자만 표기하는 링크 바로 교체
- `Link`를 이용해 `/todos/completed`로 이동

**변경 후:**
```tsx
{sections.COMPLETED.length > 0 && (
  <Link to="/todos/completed" className="... rounded-xl border ...">
    <span className="inline-block h-2.5 w-2.5 rounded-full bg-green-500" />
    {t('todos:section.completed')} ({sections.COMPLETED.length})
    <ChevronRightIcon className="h-4 w-4 ml-auto" />
  </Link>
)}
```

- `showCompleted` 상태 변수 제거 (더 이상 사용하지 않음)
- `ChevronDown` import 제거 (불필요)
