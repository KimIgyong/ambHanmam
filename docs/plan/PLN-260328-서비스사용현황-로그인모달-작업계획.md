### 단계 4: TodayPage에 모달 통합

**파일**: `apps/web/src/domain/today/pages/TodayPage.tsx`

**변경 내용**:
1. `ServiceUsageModal` import 추가
2. `useAuthStore` import 추가
3. `useEffect`에서 조건부 모달 표시 로직 추가:
   - `isMaster()` 체크
   - localStorage `amb-usage-modal-dismissed` 날짜 체크
4. `showUsageModal` state 추가
5. 모달 조건부 렌더링

```typescript
// 핵심 로직
const { isMaster } = useAuthStore();
const [showUsageModal, setShowUsageModal] = useState(false);

useEffect(() => {
  if (!isMaster()) return;
  const dismissed = localStorage.getItem('amb-usage-modal-dismissed');
  const today = new Date().toISOString().slice(0, 10);
  if (dismissed !== today) {
    setShowUsageModal(true);
  }
}, []);

const handleDismissToday = () => {
  localStorage.setItem('amb-usage-modal-dismissed', new Date().toISOString().slice(0, 10));
  setShowUsageModal(false);
};
```
