### Phase 4: 프로젝트 코멘트 (DB + 백엔드 + 프론트엔드)

**DB 변경:**
```sql
CREATE TABLE amb_project_comments (
  pjc_id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  pjt_id UUID NOT NULL REFERENCES kms_projects(pjt_id) ON DELETE CASCADE,
  usr_id UUID NOT NULL REFERENCES amb_users(usr_id),
  pjc_content TEXT NOT NULL,
  pjc_created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
  pjc_deleted_at TIMESTAMP WITH TIME ZONE
);

CREATE INDEX idx_project_comments_pjt_id ON amb_project_comments(pjt_id);
CREATE INDEX idx_project_comments_usr_id ON amb_project_comments(usr_id);
```

**백엔드 변경 파일:**
- `apps/api/src/domain/project/entity/project-comment.entity.ts` — 신규 엔티티
- `apps/api/src/domain/project/project.module.ts` — ProjectCommentEntity 등록
- `apps/api/src/domain/project/controller/project.controller.ts` — GET/POST/DELETE :id/comments
- `apps/api/src/domain/project/service/project.service.ts` — comments CRUD 메서드

**프론트엔드 변경 파일:**
- `apps/web/src/domain/project/components/ProjectCommentSection.tsx` — 신규 컴포넌트
- `apps/web/src/domain/project/components/ProjectOverviewTab.tsx` — 코멘트 섹션 추가
- `apps/web/src/domain/project/service/project.service.ts` — comments API 메서드
- `apps/web/src/domain/project/hooks/useProject.ts` — comments 쿼리키 + hooks
- `apps/web/src/locales/{ko,en,vi}/project.json` — 코멘트 관련 번역 키

**사이드 임팩트:**
- 신규 DB 테이블 → 스테이징/프로덕션에 수동 SQL 마이그레이션 필요
- ProjectEntity에 comments 관계 추가 가능하나 findById에는 불필요 (별도 쿼리)

## 4. 사이드 임팩트 분석

| 항목 | 영향도 | 설명 |
|------|--------|------|
| Overview 레이아웃 변경 | 낮음 | 편집 모드(inline input) 동일하게 동작 유지 |

```bash
ssh amb-staging "docker exec amb-postgres-staging psql -U amb_user -d db_amb -c \"
CREATE TABLE IF NOT EXISTS amb_project_comments (
  pjc_id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  pjt_id UUID NOT NULL REFERENCES kms_projects(pjt_id) ON DELETE CASCADE,
  usr_id UUID NOT NULL REFERENCES amb_users(usr_id),
  pjc_content TEXT NOT NULL,
  pjc_created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
  pjc_deleted_at TIMESTAMP WITH TIME ZONE
);
CREATE INDEX IF NOT EXISTS idx_project_comments_pjt_id ON amb_project_comments(pjt_id);
CREATE INDEX IF NOT EXISTS idx_project_comments_usr_id ON amb_project_comments(usr_id);
\""
```
