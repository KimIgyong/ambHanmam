## 1. 백엔드 변경

### 1.1 ClaudeService 쿼터 체크 추가

**파일**: `apps/api/src/infrastructure/external/claude/claude.service.ts`

**변경 내용**:
1. `BusinessException` import 추가
2. `checkQuotaIfNeeded(usageContext)` private 메서드 신규
   - `usageContext?.entId`가 없거나 `aiUsageService`가 없으면 skip
   - `aiUsageService.checkQuota(entId)` 호출
   - 초과 시 `BusinessException('E4010'|'E4011', ...)` throw
3. `sendMessage()`: `ensureClient()` 직후 `checkQuotaIfNeeded()` 호출
4. `streamMessage()`: subject async 블록 내 `ensureClient()` 직후 `checkQuotaIfNeeded()` 호출
   - catch에서 BusinessException인 경우 `{ type: 'error', error: message, code: errorCode }` 전달

### 1.2 ChatService 중복 쿼터 체크 제거

**파일**: `apps/api/src/domain/chat/service/chat.service.ts`

**변경 내용**:
- `sendMessage()` 내 `if (conversation.entId) { checkQuota... }` 블록 제거
- ClaudeService에서 일괄 처리하므로 중복

### 1.3 ClaudeStreamEvent 타입 확장

**파일**: `apps/api/src/infrastructure/external/claude/claude.service.ts`

**변경 내용**:
- `ClaudeStreamEvent` interface에 `code?: string` 필드 추가 (에러 코드 전달용)
