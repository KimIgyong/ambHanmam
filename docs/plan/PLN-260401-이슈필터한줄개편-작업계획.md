# PLAN-이슈필터한줄개편-작업계획-20260401

## 1. 시스템 개발 현황 분석
- 메인 화면: apps/web/src/domain/issues/pages/IssuesPage.tsx
- 필터 UI: apps/web/src/domain/issues/components/IssueFilters.tsx
- 공용 우측 탭: apps/web/src/shared/components/ViewScopeTab.tsx
- 번역: apps/web/src/locales/{ko,en,vi}/issues.json
- 현재는 ViewScopeTab과 좌측 이슈 범위 탭이 다른 줄에 존재

## 2. 화면 구성안

### 2.1 목표 배치
```
[My Involved] [Assigned Me] [My Issues] [All]     [My Only] [My Unit] [My Cell] [All]
└──────────── 좌측 정렬 ────────────┘     └──────────── 우측 정렬 ────────────┘
```

### 2.2 컴포넌트 구조
- 컨테이너: flex + justify-between + items-center
- 좌측 그룹: Scope 버튼 4개(rounded box)
- 우측 그룹: ViewScopeTab 그대로 재사용
- 모바일 대응: 좌측/우측 그룹이 필요 시 wrap되되, 그룹 내부 순서는 유지
