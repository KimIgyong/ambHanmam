## 2. 단계별 구현 계획

### Phase 1: 백엔드 — 개인 활동/호응 통계 API

#### 1-1. `ActivityStatsService`에 신규 메서드 2개 추가

**`getMyEngagement(entityId, userId)`** — 내 호응 통계:
```typescript
// 내가 받은 호응
receivedRatings: {
  count: number;       // 4개 rating 테이블에서 내 콘텐츠에 달린 별점 총 건수
  avgScore: number;    // 평균 점수
}
receivedReactions: {
  count: number;       // 내 이슈 댓글에 달린 이모지 총 건수
}

// 내가 반응한 호응
givenRatings: {
  count: number;       // 내가 부여한 별점 총 건수
}
givenReactions: {
  count: number;       // 내가 단 이모지 총 건수
}
```

**쿼리 전략**:
- 내가 받은 별점: `IssueRatingEntity` WHERE issue.reporter = me, `TodoRatingEntity` WHERE todo.creator = me, `CommentRatingEntity` WHERE comment.author = me, `MeetingNoteRatingEntity` WHERE note.author = me → SUM/COUNT
- 내가 받은 이모지: `IssueCommentReactionEntity` WHERE comment.author = me
- 내가 부여한 별점: 4개 rating 테이블 WHERE `usr_id = me` → COUNT
- 내가 반응한 이모지: `IssueCommentReactionEntity` WHERE `usr_id = me` → COUNT

**`getMyYesterdayActivity(entityId, userId)`** — 어제 작업 내역:
```typescript
{
  issuesCreated: number;     // 어제 내가 등록한 이슈
  issuesProcessed: number;   // 어제 상태 변경한 이슈 (status_log 기반)
  todosCreated: number;      // 어제 작성한 태스크
  notesCreated: number;      // 어제 작성한 노트
}
```

**쿼리 전략**:
- 이슈 등록: `issueRepo.count({ issReporterId: userId, issCreatedAt: Between(yesterdayStart, yesterdayEnd) })`
- 이슈 처리: `IssueStatusLogEntity` 에서 `isl_changed_by = userId AND created_at BETWEEN yesterday` → COUNT(DISTINCT iss_id)
- 태스크: `todoRepo.count({ tdoCreatedBy: userId, tdoCreatedAt: Between(...) })`
- 노트: `noteRepo.count({ mtnAuthorId: userId, mtnCreatedAt: Between(...) })`

#### 1-2. `ActivityStatsController`에 엔드포인트 2개 추가

UserEntity 조회 시 `usr_unit` 필드 추가:
```typescript
// 기존

// 변경

```

응답 매핑에 `unit` 추가:
```typescript
unit: u?.usrUnit ?? '',
```

#### 2-4. `TodayPage.tsx` — 모달 트리거 추가

기존 MASTER 모달 로직 옆에 비-MASTER 사용자용 로직 추가:
```typescript
const [showActivityModal, setShowActivityModal] = useState(false);

useEffect(() => {
  if (isMaster()) return; // MASTER는 ServiceUsageModal 사용
  const dismissed = localStorage.getItem('amb-activity-modal-dismissed');
  const today = new Date().toISOString().slice(0, 10);
  if (dismissed !== today) {
    setShowActivityModal(true);
  }
}, [isMaster]);
```
