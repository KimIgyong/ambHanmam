### Phase 1: DB 마이그레이션 + Entity + 공유 타입 (Backend Core)

**목표**: `mtn_type` 컬럼 추가, visibility 값 마이그레이션, 타입 정의 확장

#### 1-1. SQL 마이그레이션 파일

**파일**: `sql/migration_note_type.sql` (신규)

```sql
-- 1) mtn_type 컬럼 추가
ALTER TABLE amb_meeting_notes
  ADD COLUMN IF NOT EXISTS mtn_type VARCHAR(20) NOT NULL DEFAULT 'MEMO';

-- 2) 기존 데이터는 MEMO 타입 유지 (DEFAULT 'MEMO'이므로 별도 UPDATE 불필요)

-- 3) DEPARTMENT → ENTITY 마이그레이션
UPDATE amb_meeting_notes SET mtn_visibility = 'ENTITY' WHERE mtn_visibility = 'DEPARTMENT';

-- 4) 인덱스
CREATE INDEX IF NOT EXISTS idx_meeting_notes_type ON amb_meeting_notes(mtn_type);
```

const handleTypeChange = (newType: 'MEMO' | 'MEETING_NOTE') => {

  if (content && content !== '<p></p>') {

  if (newType === 'MEETING_NOTE') {

|----------|----------|--------|
