---

### Phase 1: REQ-01 - 이슈 리스트 상세보기 (난이도: 낮음)

**대상 파일**: `ProjectIssuesTab.tsx`

**구현 내용**:
1. `useState`로 `selectedIssue: IssueResponse | null` 상태 추가
2. `IssueDetailModal`, `IssueFormModal`, `IssueDeleteConfirmModal` import
3. `useCreateIssue`, `useUpdateIssue`, `useDeleteIssue` 훅 연동
4. 테이블 `<tr>`에 `onClick={() => setSelectedIssue(issue)}` 핸들러 추가
5. `cursor-pointer` 스타일 추가
6. 상세 모달, 편집 모달, 삭제 확인 모달 렌더 추가
7. 수정/삭제 후 쿼리 무효화 처리

**예상 변경 규모**: ~60줄 추가

---

### Phase 2: REQ-02 - 칸반보드 드래그앤드롭 (난이도: 낮음 - 재활용)

**대상 파일**: `ProjectKanbanTab.tsx`

**구현 내용**:
1. 기존 정적 칸반 코드 제거
2. `IssueKanbanView` 컴포넌트 import

**예상 변경 규모**: 파일 전면 교체 (~30줄로 단순화)

**주의사항**:

- `queryKey`에 `projectId` 포함하여 캐시 충돌 방지

**대상 파일**: `ProjectGanttTab.tsx`

**구현 내용**:
1. 기존 커스텀 간트 코드 제거
2. `IssueGanttView` 컴포넌트 import
3. `useQuery`로 프로젝트 이슈 로드 (기존 유지)
4. `IssueGanttView`에 `issues`와 `onIssueClick` prop 전달
5. `IssueDetailModal` 연동

**예상 변경 규모**: 파일 전면 교체 (~20줄로 단순화)

**주의사항**:

- `IssueGanttView`가 더 완성도 높은 구현이므로 이 트레이드오프는 수용 가능

### 4.2 ProjectKanbanTab.tsx (교체)

```tsx
import { useState, Suspense } from 'react';
import { useQuery } from '@tanstack/react-query';
import { projectApiService } from '../service/project.service';
import { IssueResponse } from '@amb/types';
import IssueKanbanView from '@/domain/issues/components/IssueKanbanView';
import IssueDetailModal from '@/domain/issues/components/IssueDetailModal';
// ... IssueFormModal, IssueDeleteConfirmModal

export default function ProjectKanbanTab({ projectId }) {
  const [selectedIssue, setSelectedIssue] = useState<IssueResponse | null>(null);
  
  const { data, isLoading } = useQuery({
    queryKey: ['projects', projectId, 'issues', 'kanban'],
    queryFn: () => projectApiService.getProjectIssues(projectId, { size: 200 }),
    enabled: !!projectId,
  });

  const issues = data?.data || [];

  if (isLoading) return <div>Loading...</div>;

  return (
    <>
      <IssueKanbanView issues={issues} onIssueClick={setSelectedIssue} />
      {selectedIssue && (
        <IssueDetailModal
          issue={selectedIssue}
          onClose={() => setSelectedIssue(null)}
          onEdit={...}
          onDelete={...}
        />
      )}
    </>
  );
}
```

| # | 시나리오 | 기대 결과 |
