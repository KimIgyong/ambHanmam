## 6. 배포 시 주의사항

- DB 스키마: `synchronize: true` 설정으로 서버 시작 시 자동 반영 (별도 마이그레이션 불필요)
- 기존 데이터: ent_id = NULL인 레코드는 법인 선택 후 조회되지 않음
- 프론트엔드/백엔드 모두 재빌드 및 재시작 필요
- 법인이 없는 사용자는 데이터 조회 불가 → 법인 배정 필수
