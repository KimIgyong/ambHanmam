### 1.3 외부 서비스 연동

| 서비스 | 용도 | 연동 방식 | 모듈 경로 |
|--------|------|-----------|-----------|
| **Anthropic Claude API** | AI 에이전트 대화, 자동 태깅, 지식 변환 | REST API (SDK) | `infrastructure/external/claude/` |
| **Postal Mail Server** | 이메일 발송/수신 | SMTP + Webhook | `infrastructure/external/mail/` |
| **Google Drive API** | 파일 저장/공유 | OAuth 2.0 REST API | `infrastructure/external/google-drive/` |
| **Popbill** | 전자세금계산서 발행/조회 | REST API | `infrastructure/external/popbill/` |

     │  access_token 만료 시:       │                           │

### 6.3 보안 계층 요약

| 보안 계층 | 기술 | 세부사항 |
|-----------|------|----------|
| **전송 보안** | HTTPS (TLS 1.2+) | Let's Encrypt SSL 인증서, Nginx 종단 |
| **HTTP 헤더** | Helmet | X-Frame-Options, CSP, HSTS, X-Content-Type-Options |
| **교차 출처** | CORS | 허용된 Origin만 접근 가능 |
| **속도 제한** | @nestjs/throttler | 분당 요청 수 제한 (Rate Limiting) |
| **인증** | JWT | httpOnly Cookie, access 15분 / refresh 7일 |
| **토큰 보안** | Token Rotation | 갱신 시 이전 refresh_token 무효화 |
| **비밀번호** | bcrypt | 12 rounds salt hashing |
| **데이터 암호화** | AES-256-GCM | scryptSync 키 파생, API 키 등 민감 데이터 |
| **역할 기반 접근** | RBAC | USER < MANAGER < ADMIN 3단계 |
| **입력 검증** | ValidationPipe | class-validator DTO 자동 검증 |
| **XSS 방지** | sanitize-html | HTML 콘텐츠 살균 |
| **유휴 타임아웃** | useIdleTimeout | 30분 무활동 시 자동 로그아웃 |

### 6.4 암호화 서비스 (AES-256-GCM)

### 7.3 네이밍 규칙

| 구분 | 규칙 | 예시 |
|------|------|------|
| **테이블명** | `amb_` + snake_case 복수형 | `amb_users`, `amb_conversations` |
| **PK** | `{3자 prefix}_id` (UUID v4) | `usr_id`, `cvs_id` |
| **일반 컬럼** | `{3자 prefix}_` + snake_case | `usr_email`, `cvs_title` |
| **FK** | 참조 테이블의 PK명 그대로 | `usr_id` (users 참조) |
| **Soft Delete** | `{prefix}_deleted_at` | `usr_deleted_at` |
| **생성일** | `{prefix}_created_at` | `usr_created_at` |
| **수정일** | `{prefix}_updated_at` | `usr_updated_at` |
