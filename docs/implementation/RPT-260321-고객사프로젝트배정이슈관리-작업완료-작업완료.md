## 3. 사용자 플로우

### 관리자: 프로젝트에 고객사 배정
1. `/project/projects/:id` 접속 → Overview 탭
2. 우측 사이드바 하단 "배정 고객사" 섹션 확인
3. "고객사 추가" 클릭 → Entity 고객사 드롭다운 표시
4. 고객사 선택 → 배정 완료 (토스트 알림)
5. 배정된 고객사 목록에 회사명 + 코드 표시
6. X 버튼으로 배정 해제

### 고객사 직원: 프로젝트 이슈 관리 (기존 기능 — 변경 없음)
1. `/client/login` 로그인
2. `/client/projects` 배정된 프로젝트 조회 → 이제 데이터 표시됨
3. 프로젝트 상세 → 이슈 등록/조회/코멘트/해결확인

| # | 테스트 항목 | 확인 방법 |

| 1 | 프로젝트 상세 → 사이드바에 "배정 고객사" 표시 | /project/projects/:id 접속 |
