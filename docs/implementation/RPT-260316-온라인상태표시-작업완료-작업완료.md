## 1. 구현 내용 요약

로그인 시 자동으로 사용자가 online 상태로 표시되도록, Presence 하트비트 및 SSE 훅을 **MainLayout (전역)**으로 이동하였다.

### 변경 전 (AS-IS)
- `usePresenceHeartbeat()` / `usePresenceSSE()`가 `AmoebaTalkPage.tsx`에서만 호출
- 채팅 페이지 미방문 시 항상 offline 표시

### 변경 후 (TO-BE)
- `usePresenceHeartbeat()` / `usePresenceSSE()`를 `MainLayout.tsx`에서 전역 호출
- 로그인한 모든 사용자가 어느 페이지에서든 online 표시
- 웹 PC, 모바일, PWA 모두 동일 동작
