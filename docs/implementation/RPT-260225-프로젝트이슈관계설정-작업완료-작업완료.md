## 4. API 엔드포인트

### 프로젝트 관련 (신규)

| Method | Path | 권한 | 설명 |
|--------|------|------|------|
| POST | `/api/v1/project/projects/:id/approve` | MANAGER+ | 프로젝트 승인 |
| POST | `/api/v1/project/projects/:id/reject` | MANAGER+ | 프로젝트 반려 |

| Method | Path | 권한 | 설명 |
|--------|------|------|------|

| POST | `/api/v1/migration/redmine/import` | SUPER_ADMIN | 데이터 임포트 |
| GET | `/api/v1/migration/logs` | SUPER_ADMIN | 로그 조회 |
