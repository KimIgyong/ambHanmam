## 1. 구현 내용

### 1.1 로그인 페이지 수정
- register 링크 주석처리 (초대 기반 가입만 허용)
- 로그인 성공 후 `/select-entity`로 리다이렉트

### 1.2 법인 선택 페이지 (EntitySelectPage) 신규 생성
- 소속 법인 1개: 자동 선택 후 대시보드로 바로 이동
- 소속 법인 2개 이상: 법인 카드 목록 표시, 클릭 시 해당 법인 선택 후 대시보드 이동
- 소속 법인 0개: 안내 메시지 + 로그아웃 버튼
- 국가 플래그, 법인코드, 통화 정보 표시

### 1.3 라우팅 추가
- `/select-entity` 라우트 (AuthGuard 내부, 인증 필수)

### 1.4 i18n 번역 추가 (en/ko/vi)
- selectEntity.title, subtitle, noEntity, noEntityDesc, logout

## 4. 로그인 플로우 변경

### Before
```
LoginPage → POST /auth/login → navigate('/') → MainLayout (첫번째 법인 자동 선택)
```

### After
```
LoginPage → POST /auth/login → navigate('/select-entity')
  ├─ 법인 1개 → 자동 선택 + navigate('/') → MainLayout
  └─ 법인 2개+ → 카드 선택 → navigate('/') → MainLayout
```
