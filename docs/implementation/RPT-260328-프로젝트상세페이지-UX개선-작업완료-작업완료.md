## 4. DB 마이그레이션 (배포 전 실행 필요)

Phase 4 (프로젝트 코멘트) 배포 시 `amb_project_comments` 테이블 수동 생성 필수:

```sql
CREATE TABLE IF NOT EXISTS amb_project_comments (
  pjc_id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  pjt_id UUID NOT NULL REFERENCES kms_projects(pjt_id) ON DELETE CASCADE,
  usr_id UUID NOT NULL REFERENCES amb_users(usr_id),
  pjc_content TEXT NOT NULL,
  pjc_created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
  pjc_deleted_at TIMESTAMP WITH TIME ZONE
);
CREATE INDEX IF NOT EXISTS idx_project_comments_pjt_id ON amb_project_comments(pjt_id);
CREATE INDEX IF NOT EXISTS idx_project_comments_usr_id ON amb_project_comments(usr_id);
```

### 실행 명령어
```bash
# 개발 (로컬 Docker)
docker exec amb-postgres psql -U amb_user -d db_amb -c "위의 SQL"

# 스테이징
ssh amb-staging "docker exec amb-postgres-staging psql -U amb_user -d db_amb -c '위의 SQL'"
```
