### 주요 변경 사항

1. **RichTextEditor 컴포넌트** (핵심 변경):
   - `maxHeight?: string` optional prop 추가 (하위 호환)
   - 외부 wrapper에 `flex flex-col overflow-hidden` 추가
   - 신규 스크롤 컨테이너 (`overflow-y-auto`, `overscroll-behavior: contain`) 추가
   - 툴바에 `sticky top-0 z-10 bg-white` 적용
   - EditorContent: `overflow-y-auto resize-y` 제거, Tailwind 동적 클래스 → `style={{ minHeight }}` 전환
   - Source textarea: `resize-y` 제거, `style={{ minHeight }}` 정리

2. **사용처 maxHeight 적용**:
   - Issues 폼: `maxHeight="300px"`
   - Todos 폼: `maxHeight="300px"`
   - Meeting Notes 폼: `maxHeight="400px"`
   - Notices 폼: `maxHeight="400px"`
   - Client Portal 이슈 작성: `maxHeight="400px"`
   - Today Mission (작성/수정): `maxHeight="250px"`
