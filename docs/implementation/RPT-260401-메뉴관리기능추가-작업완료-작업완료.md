- **배포 상태**: 미배포

## 1. 구현 내용 요약

- `/entity-settings/permissions` 페이지 제목을 `메뉴 및 권한 설정`으로 변경
- `메뉴관리` 탭을 첫 번째 탭으로 추가
- Entity별 메뉴 순서/카테고리 저장 기능 구현
- 사이드메뉴가 entity별 `category`와 `sortOrder`를 우선 반영하도록 수정
- 메뉴 설정 초기화 API 및 UI 추가
