## 구현 내용 요약

### 1. 별점 평가 시스템
- 이슈, 할일, 댓글, 회의록에 1~5점 별점 평가 기능
- 본인 작성 콘텐츠 셀프 평점 방지 (ForbiddenException)
- PUT (생성/수정) / DELETE (삭제) API

### 2. 활동지수 (Activity Score)
- 콘텐츠 생성 기반 가중치 점수
- 기본 가중치: 이슈(5), 회의록(4), 댓글(4), 할일(2), 채팅(1)
- 부서별 가중치 커스터마이징 가능

### 3. 참여지수 (Engagement Score)
- 받은 별점 합계 + 리액션 수 기반 점수
- 참여 가중치 별도 설정 가능

### 4. 일별 스냅샷
- `@Cron('0 1 * * *')` 매일 01:00 자동 집계
- 수동 집계 API 제공 (POST `/activity-stats/aggregate`)

### 5. 관리자 페이지
- 멤버별 통계 테이블 (랭킹, 활동점수, 참여점수, 총점)
- 가중치 설정 에디터
- 경로: `부서 설정 > 활동지수 관리`

### 6. 글로벌 StarRating 컴포넌트
- 기존 회의록용 StarRating을 글로벌 컴포넌트로 통합
- 평균 점수, 평가 수, 본인 콘텐츠 여부 표시
- 반별(half-star) 비주얼 지원
