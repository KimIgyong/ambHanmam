## 1. 구현 내용 요약

로그인한 사용자가 `/documents` 문서함 페이지에서 현재 탐색 중인 Google Drive 폴더에 파일을 업로드할 수 있는 기능을 구현했다.

### 주요 기능
- 폴더 내부 진입 시 "업로드" 버튼 표시
- 다중 파일 동시 선택 지원
- 파일별 업로드 진행 상태 표시 (대기/업로드 중/완료/실패)
- 50MB 파일 크기 제한 (프론트엔드 + 백엔드 양쪽 검증)
- 업로드 완료 후 파일 목록 자동 갱신
- 5초 후 업로드 큐 자동 제거

## 3. API 엔드포인트

### `POST /api/v1/drive/upload`
- **인증**: Bearer Token (로그인 필수)
- **Content-Type**: `multipart/form-data`
- **파라미터**:
  - `file`: 업로드 파일 (최대 50MB)
  - `folder_id`: 대상 Google Drive 폴더 ID
- **응답**: `{ success, data: { fileId, name, mimeType, webViewLink, size }, timestamp }`
