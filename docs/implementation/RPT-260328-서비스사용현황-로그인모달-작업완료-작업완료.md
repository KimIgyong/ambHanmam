## 1. 구현 내용

Entity MASTER 사용자가 로그인 후 TodayPage에 진입하면 자동으로 서비스 사용현황 모달을 표시합니다.

### 모달 내용
- **직원 현황**: 총 활성 직원 수, 최근 7일 활성 사용자 수/비율
- **AI 사용 현황**: 월간/일일 쿼터 프로그레스 바, 누적/오늘 토큰 사용량
- **쿼터 경고**: quotaStage에 따른 색상별 경고 메시지 (CAUTION/WARNING/SUSPENDED)
- **액션**: 오늘 안 보기, 토큰 충전하기, 상세 사용량 보기

### 표시 조건
- `user.role === 'MASTER'`인 경우만 표시
- 1일 1회 제한 (localStorage `amb-usage-modal-dismissed` 키 기반)
- 닫기: X 버튼, 오버레이 클릭, ESC 키

### 백엔드 변경
- **없음** — 기존 API 3개 재사용:
  - `GET /entity-settings/usage` → AI 사용량/쿼터
  - `GET /entity-settings/members` → 직원 수
  - `GET /entity-settings/work-statistics/overview` → 7일 활동 통계
