# RPT-260413 이슈 코멘트 고객사 공개 제어

## 1. 구현 내용 요약

이슈 코멘트에 **고객사(CLIENT_LEVEL) 공개 여부 제어** 기능을 구현했다.

- 내부 사용자(USER_LEVEL)가 코멘트 작성 시 "고객사에게 공개" 체크박스로 공개 여부 선택
- 기존 코멘트의 공개/비공개를 Eye/EyeOff 아이콘 버튼으로 토글 가능
- CLIENT_LEVEL 사용자가 이슈 상세 조회 시 **공개 코멘트만 표시** (서버 필터링)
- CLIENT_LEVEL 사용자가 작성하는 코멘트는 **자동 공개**
- 대댓글은 부모 코멘트의 공개 상태를 상속하며, 부모 토글 시 대댓글도 동기 전환

## 3. DB 변경사항

```sql
ALTER TABLE amb_issue_comments 
  ADD COLUMN isc_client_visible BOOLEAN NOT NULL DEFAULT false;

-- 기존 CLIENT_LEVEL 사용자 작성 코멘트만 공개 전환
UPDATE amb_issue_comments 
SET isc_client_visible = true 
WHERE isc_author_id IN (
  SELECT usr_id FROM amb_users WHERE usr_level_code = 'CLIENT_LEVEL'
);
```
