### TC-1.1: Drive 미설정 시 로컬 업로드 폴백
| 항목 | 내용 |
|------|------|
| **전제조건** | Google Drive 설정이 되어있지 않은 법인 |
| **절차** | 1. 이슈 생성/편집 모달 오픈 → 2. 업로드 탭 선택 → 3. 파일 드래그 또는 클릭 업로드 |
