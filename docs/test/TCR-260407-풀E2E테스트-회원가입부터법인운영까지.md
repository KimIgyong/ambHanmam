# TCR-260407 Full E2E Test — 회원가입부터 법인 운영까지

> **테스트 대상**: Portal (www.amoeba.site) + AMA (ama.amoeba.site)
> **테스트 환경**: 스테이징 (stg-www.amoeba.site / stg-ama.amoeba.site) → 프로덕션
> **작성일**: 2026-04-07

---

## 테스트 플로우 개요

```
[Phase 1] 포털 회원가입
  www.amoeba.site → 무료체험 → 사용자등록 → 이메일인증 → 완료화면 → 임시비번 확인

[Phase 2] AMA 최초 로그인
  ama.amoeba.site/{entityCode}/login → 임시비번 로그인 → 비밀번호 변경 → 대시보드 진입

[Phase 3] 법인관리자 운영 (MASTER)
  조직 설정: 유닛 추가 → 셀 추가
  사용자 관리: 사용자 초대 → 유닛/셀 배정
  고객사 관리: 고객사 생성 → 고객사 담당자 초대
  외부 연동: 구글 드라이브 설정
  커스텀앱: 아메바톡 연결(링크), 레드마인 연결(iframe/jwt)
  앱스토어: 차량관리앱 등록, 세일즈리포트앱 등록

[Phase 4] 포털 사용자 (구독 관리)
  www.amoeba.site 로그인 → 마이포털 → 구독정보 확인(Free Tier)
  유료 전환: 토큰/스토리지 업그레이드 기능 동작 확인
```
