## TC-01: 로그인 시 온라인 상태 전환

| 항목 | 내용 |
|------|------|
| **전제조건** | 사용자 A, B 두 계정 준비 |
| **절차** | 1. 사용자 A 로그인 → Today 페이지 접속<br>2. 사용자 B 로그인 → Amoeba Talk 접속<br>3. 사용자 B의 DM 목록 또는 Crew 섹션에서 사용자 A 상태 확인 |
| **기대결과** | 사용자 A가 채팅 페이지가 아닌 Today에 있어도 **녹색 점(online)** 표시 |
| **결과** | ✅ Pass |
