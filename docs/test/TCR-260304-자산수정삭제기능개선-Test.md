# 테스트 케이스 (TC)

- 문서명: 자산 수정/삭제 기능 개선 테스트
- 작성일: 2026-03-04

## 단위/기능 테스트
1. 자산 상세 > 전체수정 > 저장 시 `PATCH /assets/:id` 호출 성공
2. 카테고리를 회의실 외 값으로 변경 시 회의실 전용 필드 초기화
3. 자산 상세 > 삭제 클릭 시 확인 후 `DELETE /assets/:id` 호출 성공
4. 수정/삭제 후 자산 목록 React Query invalidate로 갱신
5. 기존 상태변경(`PATCH /assets/:id/status`) 정상 동작 유지

## 통합 시나리오
1. 자산 등록 → 상세 → 전체수정(이름/위치/금액 변경) → 재조회 반영 확인
2. 자산 삭제 후 목록 미노출 확인(soft delete 필터)

## 검증 결과
- `npm run build` 통과
