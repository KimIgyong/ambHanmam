## 1. 개요

구독 서비스는 법인(Entity) 단위로 AI 토큰, 스토리지, 사용자 수를 관리하는 SaaS 과금 시스템입니다. 3개의 요금제(FREE / BASIC / PREMIUM) 중 선택하여 사용하며, 토큰과 스토리지는 필요에 따라 추가 구매할 수 있습니다.

### 핵심 개념

| 개념 | 설명 |
|------|------|
| **AI 토큰** | AI 채팅, 번역, 분석 등 AI 기능 사용 시 차감되는 단위 |
| **스토리지** | 파일 업로드, Google Drive 연동 등에 사용되는 저장 공간 |
| **결제 주기** | 월간(Monthly) 또는 연간(Annual) 선택 가능 (BASIC 이상) |

---

## 2. 접속 경로

**사이드바 > 법인 설정 > 구독 관리**

- 경로: `/entity-settings/subscription`
- 접근 권한: 법인 관리자(MASTER) 이상
- 아이콘: 💳 (CreditCard)

## 3. 요금제 (Plan)

### 3.1 요금제 비교

| 항목 | FREE | BASIC | PREMIUM |
|------|------|-------|---------|
| **가격** | 무료 | $2/user/월 | 별도 문의 |
| **사용자 수** | 최대 5명 | 6~49명 (5명 단위) | 50명 이상 |
| **무료 포함 사용자** | 5명 | 5명 | 5명 |
| **AI 토큰** | 50,000 (1회 지급) | 20,000/user/월 | 별도 협의 |
| **토큰 월 리셋** | ❌ | ✅ 매월 1일 자동 충전 | ✅ |
| **기본 스토리지** | 1 GB | 3 GB | 별도 협의 |
| **최대 스토리지** | 1 GB (추가 불가) | 100 GB | 별도 협의 |
| **스토리지 추가** | ❌ | 5 GB 단위, $1/GB | ✅ |
| **토큰 추가 구매** | 10,000 단위, $1 | 50,000 단위, $1 | ✅ |
| **연간 결제** | ❌ | ✅ (2개월 무료) | ✅ |
| **추천 보상** | 10명 초대 → 50,000 토큰 | 10명 초대 → 50,000 토큰 | ✅ |

### 3.2 토큰 차감 순서

AI 기능 사용 시 토큰은 아래 순서로 차감됩니다:

1. **기본(BASE)** 토큰 — 플랜에 포함된 토큰
2. **추가(ADDON)** 토큰 — 별도 구매한 토큰
3. **추천(REFERRAL)** 토큰 — 추천 보상으로 받은 토큰

### 3.3 스토리지 사용 정책

| 사용률 | 상태 | 동작 |
|--------|------|------|
| 0~79% | 정상 | 파란색 프로그래스바 |
| 80~99% | 주의 | 노란색 프로그래스바 |
| 100~119% | 경고 | 주황색 프로그래스바 + 경고 배너. 업로드는 가능 |
| 120% 이상 | 차단 | 빨간색 프로그래스바 + 차단 배너. **업로드 불가** |

> 💡 업로드가 차단되면 스토리지를 추가 구매하거나 기존 파일을 삭제하여 사용량을 줄여야 합니다.

## 4. 플랜 탭

구독 관리 페이지의 첫 번째 탭입니다.

### 4.1 화면 구성

| 영역 | 설명 |
|------|------|
| **현재 플랜 카드** | 플랜 이름, 상태 배지, 결제 주기, 사용자 수, 시작일, 다음 결제일 |
| **액션 버튼** | BASIC 업그레이드 / 구독 해지 |
| **플랜 상세** | 가격, 토큰 정책, 스토리지 정책, 최대 사용자 수 |

### 4.2 상태 배지

| 상태 | 색상 | 설명 |
|------|------|------|
| `ACTIVE` | 🟢 초록 | 정상 구독 중 |
| `TRIALING` | 🔵 파랑 | 체험 기간 (결제 전) |
| `PAST_DUE` | 🟡 노랑 | 결제 지연 |
| `CANCELED` | 🔴 빨강 | 해지됨 |

### 4.3 BASIC 업그레이드 방법

1. 플랜 탭에서 **[BASIC 업그레이드]** 버튼 클릭
2. Polar.sh 결제 페이지가 새 탭에서 열립니다
3. 카드 정보 입력 후 결제 완료
4. 자동으로 BASIC 플랜이 활성화됩니다
5. 기본 토큰(유저 수 × 20,000)이 지급되고, 스토리지가 3GB로 확장됩니다

### 4.4 구독 해지

1. 플랜 탭에서 **[구독 해지]** 버튼 클릭
2. 확인 대화상자에서 **확인** 클릭
3. 현재 결제 기간 만료까지 서비스 이용 가능
4. 기간 만료 후 FREE 플랜으로 자동 전환

> ⚠️ 해지 후 남은 추가 구매 토큰(ADDON)은 소멸되지 않습니다. 단, BASE 토큰은 월 리셋 시 만료됩니다.

## 5. AI 토큰 탭

두 번째 탭으로, AI 토큰 잔액과 지갑별 상세를 확인합니다.

### 5.1 화면 구성

| 영역 | 설명 |
|------|------|
| **총 잔액 카드** | 전체 토큰 잔액 (인디고-퍼플 그라데이션) |
| **지갑 카드 ×3** | BASE / ADDON / REFERRAL 각각 잔액, 누적 지급, 누적 사용 |
| **토큰 충전** | 충전 수량 선택 + 구매 버튼 |

### 5.2 지갑 유형

| 유형 | 색상 | 충전 방식 |
|------|------|-----------|
| **BASE** | 🔵 파랑 | FREE: 가입 시 1회(50,000). BASIC: 매월 1일 자동 충전 |
| **ADDON** | 🟣 보라 | 토큰 추가 구매 시 입금 |
| **REFERRAL** | 🟡 노랑 | 추천 보상 달성 시 자동 입금 |

### 5.3 토큰 추가 구매

1. 토큰 탭의 **토큰 충전** 영역에서 수량 선택 (10,000 / 50,000 / 100,000 / 500,000)
2. **[충전]** 버튼 클릭
3. Polar.sh 결제 페이지에서 결제 완료
4. ADDON 지갑에 토큰이 즉시 입금됩니다

### 5.4 월 리셋 (BASIC 플랜)

- 매월 1일 00:00(KST)에 자동 실행
- 기존 BASE 잔액은 만료 처리 (DEBIT 레코드 생성)
- 새 BASE 토큰 지급: `유료 사용자 수 × 20,000`
- ADDON, REFERRAL 토큰은 리셋되지 않음 (잔액 유지)

### 5.5 토큰 소진 시

AI 토큰이 0이 되면:
- AI 채팅 전송 시 **"AI service suspended"** 에러가 표시됩니다
- 토큰을 추가 구매하거나, 다음 월 리셋까지 대기해야 합니다
- 토큰 잔액이 0이어도 AI 외 기능(할일, 이슈, 프로젝트 등)은 정상 사용 가능합니다

## 6. 스토리지 탭

세 번째 탭으로, 스토리지 사용 현황과 추가 구매를 관리합니다.

### 6.1 화면 구성

| 영역 | 설명 |
|------|------|
| **경고/차단 배너** | 사용량 100% 이상 시 노란 경고, 120% 이상 시 빨간 차단 |
| **사용 현황 카드** | 프로그래스바 + 사용량(GB) / 총량(GB) (%) |
| **용량 분류** | 기본 스토리지 / 추가 스토리지 / 총 할당량 |
| **스토리지 추가** | 추가 용량 선택 + 구매 버튼 |

### 6.2 스토리지 추가 구매

1. 스토리지 탭의 **스토리지 추가** 영역에서 용량 선택 (5 / 10 / 20 / 50 / 100 GB)
2. **[추가]** 버튼 클릭
3. Polar.sh 결제 페이지에서 결제 완료
4. 추가 스토리지에 즉시 반영됩니다

> ⚠️ FREE 플랜은 스토리지 추가 구매가 불가합니다. BASIC 이상으로 업그레이드 후 구매하세요.

### 6.3 스토리지 사용량에 포함되는 항목

- 파일 업로드 (문서함, 이슈 첨부 등)
- Google Drive 연동 파일

## 7. 결제 프로세스

### 7.1 결제 흐름

```
[구매 버튼 클릭]
    ↓
[AMA 서버에서 Polar.sh Checkout URL 생성]
    ↓
[Polar.sh 결제 페이지 (새 탭)]
    ↓
[카드 정보 입력 → 결제 완료]
    ↓
[Polar.sh → AMA 웹훅 전송]
    ↓
[자동 처리: 플랜 활성화 / 토큰 입금 / 스토리지 확장]
```

### 7.2 지원 결제 수단

- 신용카드 / 체크카드 (Visa, Mastercard, AMEX 등)
- Polar.sh를 통한 안전한 결제 처리

### 7.3 결제 실패 시

- 결제 페이지에서 에러 메시지가 표시됩니다
- 구독 관리 페이지로 돌아와 다시 시도하세요
- 반복 실패 시 카드사에 문의하거나, 다른 카드로 재시도하세요

## 8. FAQ

### Q. 법인을 새로 생성하면 어떤 플랜이 적용되나요?
신규 법인은 **FREE 플랜**이 자동 부여되며, 50,000 AI 토큰과 1GB 스토리지가 제공됩니다.

### Q. BASIC 플랜에서 사용자 수를 변경할 수 있나요?
초기 업그레이드 시 사용자 수를 선택합니다. 최소 6명, 최대 49명이며 5명 단위로 설정됩니다. 변경이 필요한 경우 관리자에게 문의하세요.

### Q. 연간 결제의 혜택은 무엇인가요?
BASIC 연간 결제 시 **2개월 무료** 혜택이 적용됩니다 (12개월 가격으로 14개월 사용).

### Q. 토큰이 부족해도 기존 업무를 사용할 수 있나요?
네. 토큰 소진 시 **AI 기능만** 일시 중단됩니다. 할일, 이슈, 프로젝트, 채팅, 파일 업로드 등 AI 외 기능은 정상 사용할 수 있습니다.

### Q. 추가 구매한 토큰은 월 리셋 시 사라지나요?
아니요. **ADDON 토큰과 REFERRAL 토큰은 월 리셋 대상이 아닙니다.** 소진할 때까지 유지됩니다. 월 리셋은 BASE 토큰에만 적용됩니다.

### Q. 스토리지가 120%를 넘으면 기존 파일이 삭제되나요?
아니요. **기존 파일은 삭제되지 않습니다.** 다만 새로운 파일 업로드가 차단됩니다. 스토리지를 추가 구매하거나 불필요한 파일을 삭제하면 업로드가 다시 가능해집니다.

### Q. 구독을 해지하면 데이터가 사라지나요?
아니요. 구독 해지 후에도 데이터는 유지됩니다. 단, FREE 플랜으로 전환 시 AI 토큰 월 리셋이 중단되고, 스토리지 한도가 1GB로 변경됩니다(기존 초과 데이터는 유지되나 추가 업로드 불가).

### Q. 결제 영수증은 어디서 확인하나요?
결제 완료 시 입력한 이메일로 Polar.sh에서 영수증이 자동 발송됩니다.
