## 1. HBA 전체 메뉴 구조

HBA는 **2-영역 하이브리드 네비게이션** 구조를 채택합니다.

- **상단 GNB** → HANMAM 업무모듈 (경영정보·영업·서비스·지원·알림·자금) + 업무도구(AMA) 진입점
- **좌측 LNB** → 업무도구(AMA) 클릭 시 AMA 사이드바 슬라이드 출현 (또는 AMA 영역 전환)

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│  GNB (상단 헤더)                                                                  │
│  🏢 HANMAM │ 경영정보 │ 영업업무 │ 서비스업무 │ 업무지원 │ 알림게시판 │ 자금업무 │ 💡업무도구(AMA) │
├─────────────────────────────────────────────────────────────────────────────────┤
│               메인 콘텐츠 영역 (선택된 메뉴에 따라 변화)                           │
└─────────────────────────────────────────────────────────────────────────────────┘
```
